# 多分支和第二问时间使用示例

本文档演示如何在剧本中使用**多分支选择**和**第二问时间**功能。

## 1. 多分支选择（ChoiceLine）

### 基本用法

在片段中添加 `type: 'choice'` 的行，提供多个选项供玩家选择：

```typescript
{
  id: 'EXAMPLE-01',
  time: '09:00',
  description: '示例：选择分支',
  loop: 'A',
  unlockFlags: [],
  lines: [
    {
      type: 'narration',
      text: '你站在十字路口，不知道该往哪里走。'
    },
    {
      type: 'choice',
      choices: [
        {
          text: '向左走',
          targetSegmentId: 'EXAMPLE-02',
          setFlag: 'went_left'
        },
        {
          text: '向右走',
          targetSegmentId: 'EXAMPLE-03',
          setFlag: 'went_right'
        },
        {
          text: '站在原地',
          targetSegmentId: 'EXAMPLE-04'
          // 不设置 flag，表示这个选择不会解锁新内容
        }
      ]
    }
  ]
}
```

### 选择分支说明

- **text**: 选项显示的文本
- **targetSegmentId**: 选择后跳转到的片段 ID
- **setFlag** (可选): 选择后设置的 flag，可用于解锁其他片段

### 对应的目标片段

```typescript
// 向左走的结果
{
  id: 'EXAMPLE-02',
  time: '09:05',
  description: '向左走的结果',
  loop: 'A',
  unlockFlags: ['went_left'], // 需要选择了"向左走"才能看到
  lines: [
    {
      type: 'narration',
      text: '你选择了向左走。'
    },
    {
      type: 'dialogue',
      text: '这条路看起来很熟悉...'
    }
  ]
}

// 向右走的结果
{
  id: 'EXAMPLE-03',
  time: '09:05',
  description: '向右走的结果',
  loop: 'A',
  unlockFlags: ['went_right'],
  lines: [
    {
      type: 'narration',
      text: '你选择了向右走。'
    },
    {
      type: 'dialogue',
      text: '这里似乎有什么不对劲...'
    }
  ]
}
```

## 2. 第二问时间（secondTime）

### 基本用法

使用 `secondTime` 字段实现"两问一锁"机制：需要先输入第一个时间，再输入第二个时间，才能解锁特定片段。

```typescript
// 第一个片段：需要先看到这个
{
  id: 'P01-01',
  time: '08:20',
  description: 'A 发约楼顶的消息',
  loop: 'C',
  unlockFlags: ['loop_b_complete'],
  lines: [
    {
      type: 'narration',
      text: '手机屏幕上显示一条消息：'
    },
    {
      type: 'dialogue',
      character: 'A',
      text: '等会儿上去聊聊。'
    }
  ]
}

// 第二个片段：需要先输入 08:20，再输入 10:07 才能看到
{
  id: 'P01-04',
  time: '10:07',           // 第一问时间
  secondTime: '08:20',     // 第二问时间
  description: 'A 被推下（真相版）',
  loop: 'D',
  unlockFlags: ['loop_c_complete', 'saw_p01-01'], // 还需要先看过 P01-01
  lines: [
    {
      type: 'narration',
      text: '楼顶。清晰的记忆。'
    },
    {
      type: 'dialogue',
      character: 'A',
      text: '我们分手吧。'
    },
    {
      type: 'narration',
      text: '我抓住她的手腕，有意识地...多推了一点点。'
    }
  ]
}
```

### 工作流程

1. 玩家第一次输入 `10:07` → 显示普通版本（如果有的话）
2. 玩家输入 `08:20` → 显示 P01-01 片段
3. 玩家再次输入 `10:07` → 系统检测到需要第二问，显示第二问提示
4. 玩家输入 `08:20` → 系统匹配到 `time: '10:07'` + `secondTime: '08:20'` → 显示 P01-04 真相版

### 在开场片段中使用第二问

```typescript
export const startSegment: ScriptSegment = {
  id: 'START',
  time: 'START',
  description: '游戏开场',
  loop: 'START',
  unlockFlags: [],
  lines: [
    {
      type: 'narration',
      text: '时间是……？'
    },
    {
      type: 'dialogue',
      text: '不，是他在问我。'
    },
    {
      type: 'input',
      placeholder: '输入时间 (HH:MM)',
      secondQuestionText: '我记得，那天是从几点开始不对劲的？' // 第二问提示文本
    }
  ]

```

## 3. 完整示例：结合使用

```typescript
// 场景：玩家需要先看到某个片段，然后通过选择分支决定后续剧情

// 第一步：初始片段（需要特定 flag 解锁）
{
  id: 'SCENE-01',
  time: '14:00',
  description: '发现线索',
  loop: 'B',
  unlockFlags: ['found_clue'], // 需要先找到线索
  lines: [
    {
      type: 'narration',
      text: '你发现了一张纸条。'
    },
    {
      type: 'dialogue',
      text: '上面写着：{bold}今晚8点，老地方见{/bold}'
    },
    {
      type: 'choice',
      choices: [
        {
          text: '去赴约',
          targetSegmentId: 'SCENE-02',
          setFlag: 'decided_to_meet'
        },
        {
          text: '不去',
          targetSegmentId: 'SCENE-03',
          setFlag: 'decided_not_to_meet'
        }
      ]
    }
  ]
}

// 第二步：如果选择"去赴约"，使用第二问时间解锁真相
{
  id: 'SCENE-02',
  time: '20:00',        // 第一问：晚上8点
  secondTime: '14:00',  // 第二问：下午2点（发现纸条的时间）
  description: '赴约的真相',
  loop: 'C',
  unlockFlags: ['decided_to_meet', 'saw_scene_01'], // 需要选择了"去赴约"且看过 SCENE-01
  lines: [
    {
      type: 'narration',
      text: '你准时到达了约定地点。'
    },
    {
      type: 'dialogue',
      text: '但是...{delay:1}没有人。'
    },
    {
      type: 'narration',
      text: '你意识到，这可能是一个陷阱。'
    }
  ]
}

// 第三步：如果选择"不去"
{
  id: 'SCENE-03',
  time: '20:00',
  description: '不赴约的结果',
  loop: 'C',
  unlockFlags: ['decided_not_to_meet'],
  lines: [
    {
      type: 'narration',
      text: '你决定不去赴约。'
    },
    {
      type: 'dialogue',
      text: '也许这是正确的选择...'
    }
  ]
}
```

## 4. 注意事项

1. **Flag 管理**：
   - Flag 是字符串，建议使用有意义的名称（如 `'went_left'`, `'found_clue'`）
   - Flag 一旦设置就不会自动清除，需要手动管理

2. **解锁条件**：
   - `unlockFlags` 数组中的所有 flag 都必须存在才能解锁片段
   - 空数组 `[]` 表示无需解锁条件

3. **第二问时间**：
   - 只有设置了 `secondTime` 的片段才会触发第二问
   - 第二问会显示 `InputLine` 中的 `secondQuestionText` 提示文本

4. **片段 ID**：
   - 必须唯一
   - 建议使用有意义的命名（如 `'P01-04'`, `'SCENE-02'`）

5. **选择分支**：
   - 选择分支会立即跳转到目标片段
   - 如果目标片段有 `unlockFlags`，需要确保 flag 已设置

## 5. 测试建议

1. 测试选择分支：确保每个选项都能正确跳转
2. 测试第二问：确保输入两个时间后能正确匹配到对应片段
3. 测试 Flag 解锁：确保 flag 正确设置后能解锁新片段
4. 测试组合场景：测试选择分支 + 第二问时间的组合使用
