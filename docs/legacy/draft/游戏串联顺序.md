### 排序这一步，我们先定“原则”，再给一套具体串联草案

先不急着列完整树状图，先把**叙事排序的目标**钉死：

- **前期（Loop 1–2）**：玩家只知道“现在的 B”，A 和旧案只是模糊传闻 → 玩家默认自己是“失去她的受害者”。  
- **中期（Loop 3–4）**：  
  - 先强化“B 失踪”的紧张 + E 的温柔线；  
  - 再用“假闪回”+ 代词模糊引出旧案，让玩家以为“当年是意外 or 替罪羊是凶手”。  
- **后期（Loop 5–6）**：  
  - 通过两问一锁逐步拆开：先 A 真相，再 B 真相，再 E 共犯；  
  - 让玩家意识到是“两次杀人 + 多个人争夺 C 位置”。  
- **终局（Loop 7+）**：  
  - meta 问现实时间；  
  - E 顶罪的后事片段；  
  - 主角把 E 视为“最后的 C”，时间仍停在 10:37。

再在这个骨架上做**多支路的片段排序**：不是“线性一条路”，而是每个 Loop 给玩家一组“明显被提示的时间”，再加一些“自由撞出来的时间”。

### 一、整体结构（Loop 视角，而不是严格“第几周目”）

你可以在 `游戏串联顺序.md` 里先放一个这样的概览：

- **Loop A：完全伪日常**  
- **Loop B：B 消失后的日常错位**  
- **Loop C：假闪回与旧案影子**  
- **Loop D：A 真相 & 初步接触 B 案当天**  
- **Loop E：B 真相 & E 共犯**  
- **Loop F：E 顶罪 & 后事**  
- **Loop G：meta 终局 & 可能的再次循环**

每个 Loop 内部是「若干时间输入 → 对应若干片段」，玩家不一定全部看完才进入下一 Loop，但 **关键信息用 flag 控制**，一旦满足条件就允许进入下一个层级。

---

### 二、具体串联草案（可直接抄到 `游戏串联顺序.md` 打磨）

下面我用“Loop X”+“推荐会看到的时间/片段”来写，你可以看成脚手架。

---

#### Loop A：完全伪日常（让玩家以为是校园恋爱）

**目标**：只暴露 P03 组的“B 线伪日常”，不提 A，也不提 B 失踪。

**此 Loop 内推荐可见时间 → 片段：**

- `08:05` → P03-01（和 B 一起上学）  
- `12:10` → P03-02（教室午餐，B+D+E 同场）  
- `18:40` → P03-03（垃圾袋 + 锯子·伪日常版）  
- `23:15` → P03-04（夜间独白·软版，“时间是……”半句）

**Loop A 结束条件**（系统内部）：  
- 玩家至少看过一次 `08:05` 和一次 `12:10`，确认“B 存在 & 和主角关系升温”。  
- 可选：如果玩家乱输 `10:37` / `11:00` 等敏感时间，先统一导向一个「记忆空白片段」，不让他们过早进入闪回。

---

#### Loop B：B 消失后的日常错位（让“空位”先吓人）

**目标**：同样的日常时间，再看一遍，但 B 不在了；让玩家自然认为“这是 B 失踪案”。

**此 Loop 内时间 → 片段：**

- `08:05` → P06-01（空路，D 不安）  
- `12:10` → P06-02（空座位 + 便利贴）  
- `18:40` → P03-03 或 P03-03 的轻微变化版（再看一次伪日常垃圾袋）  
- `23:15` → P06-04（D 或 E 来敲门，问 B 的事/什么都没说）

**Loop B 同时埋下几个“未来时间”：**

- D 在 `12:10` 的对话里提到：“那天她说，**下午三点一十**要来教室找你。” → 暗示 `15:10`。  
- 某个同学在闲聊中提到：“一年前的那天早上，好像是 **八点二十**的课间出事的。” → 暗示 `08:20`。

**Loop B 结束条件：**

- 玩家至少看过一次 `08:05` 的空路 & `12:10` 的空座；  
- 获得至少一个具体时间线索（比如文本中出现 `15:10` 或 `08:20`）。

---

#### Loop C：假闪回与旧案影子（时间开始“回过去”，但骗你是意外）

**目标**：让玩家第一次“看见楼顶和问话室”，但看到的是加工过的假记忆，仍能维持“主角是受害者”的错觉。

**在这一 Loop 中，开放这些时间的特殊处理：**

- `10:37`（第一次命中） → P01-03（A 案假闪回：“我没拉住她”版）  
- `11:00`（第一次命中） → P01-06（旧案问话·模糊版，仅用代词，不说名字）  
- `08:20` → P01-01（A 发“等会儿上去聊聊”的短信）  
- `23:15` → P02-03（旧案后失眠的夜晚，10:37 vs 23:15 撞在一起）

依然保持：  
- `08:05` / `12:10` / `18:40` 在这一阶段仍可以回看 B 失踪前后的日常版本；  
- 只是玩家心中开始出现“原来一年前还有一个她”的模糊轮廓。

**Loop C 结束条件：**

- 玩家看过 `10:37` 的假闪回；  
- 看过至少一个旧案相关片段（`08:20` 或 `11:00`）；  
- 心里有了“以前也有一个同名的她”的印象。

---

#### Loop D：A 真相 & 初步接触 B 案“当天”

**目标**：通过两问一锁，把 `10:37` / `11:00` 的其中一条组合解开，先让玩家看到 A 案真相，但暂时还不完全意识到 B 也是同样模式。

**开放：**

- `10:37` + 第二问 `08:20` → P01-04（A 被推下真相版）  
- `10:08` → P01-05（A 死后 1 分钟，看表段）  
- `11:00` + 第二问（例如 `09:30`） → P02-01（第二轮问话，把旧替罪羊钉死）  

此时仍然控制：  
- B 案当天的 `10:37` 组合（比如 `10:37` + `15:10`）暂时 **不开放**；  
- `18:40` 真相版也先关着，只允许玩家看伪日常垃圾袋。

**Loop D 里还能继续插入几段“现在”的片段：**

- `15:10`（现在） → P04-02/D 担心 B（不是当日，还只是酝酿）  
- `18:10`（现在） → P04-03/E 和主角散步，试图“拯救”。

**Loop D 结束条件：**

- 玩家至少解锁一次 A 真相版 10:37（P01-04）；  
- 看过一次 `10:08` 或 `23:15` 的看表崩溃片段。

---

#### Loop E：B 真相 & E 共犯（重演第二次杀人）

**目标**：在玩家已经知道“一年前是他杀的 A”之后，再揭露“这次 B 也是同样模式”，并引出 E 共犯分尸。

**开放 B 案相关的第二套组合：**

- `10:37` + 第二问 `15:10` → P05-03（B 被杀真相版）  
- `10:05` → P05-02（等待 B 的焦虑，看表）  
- `10:08/10:09/10:10` → P05-04（B 案后几分钟，快速进入处理模式的切片）  
- 某个时间（如 `16:20`） → P05-05（E 清晰目击 B 案，从远处看到全过程）  
- `18:39` → P05-06（分尸前一分钟看表）  
- `18:40` → P05-07（E 出现，说 C，帮忙处理）  
- `18:55` → P05-08（洗手/洗锯子，日常化处理）

此时 `08:05/12:10` 再看一遍，玩家已经知道：  
- 曾经和现在的“她”不是同一个人；  
- 主角的温柔日常都是建在两次杀人的基础上。

**Loop E 结束条件：**

- 玩家看过 B 真相 10:37；  
- 看过 18:40 真相；  
- 至少一段 E 的目击或共犯视角。

---

#### Loop F：E 顶罪 & 后事（现实收束）

**目标**：通过极少量结局片段，告诉玩家现实如何处理这两起案子 → E 被写成“真凶”，主角“无罪”。

**开放：**

- 某课间（如 `10:30`） → P07-01（关于 E 的流言）  
- 某问话时间（如 `11:30`） → P07-02（G 对主角说“你可以回去了”）  
- `14:30` → P08-01（探视室，E 自称 C）  
- `18:10` → P08-02（回家路上的新闻推送 + 看表）  

玩家在这些段里能感到：  
- 替罪羊从“那家伙”变成了 E；  
- 主角被官方、老师、同学“正式放过”；  
- 但他的时间还卡在 10:37 上。

---

#### Loop G：meta 终局 &（可选）循环

**目标**：最后一次把“时间是？”从故事内拉到现实，把玩家也拉进加害链，然后可选择是否“假重启”。

**开放：**

- meta 特殊输入 → P09-01（让玩家输入现实时间，她纠正，指出“你又在 XX:XX 看这一切”）  
- 可选：回到开场 `时间是……？` 文本，但略改一句话，形成软循环 → P09-02

### 三、如何保持“多支路”，而不是一条死线

虽然整体上我们按 Loop A→G 设计信息开放节奏，但实际游玩可以通过：

- **flag 控制开放程度，而不是强制周目数**：  
  - 例如：只要玩家看过某个片段，就允许某类组合出现，不必规定“必须第 N 周目”。

- **每个 Loop 给多个“推荐入口”，而不是只给一个**：  
  - 比如在 Loop C 中，同时开放 10:37 假闪回 + 11:00 假问话 + 08:20 短闪回，玩家可以任意顺序看；  
  - 顺序不同，误导方式略有差异，但总会收束回我们设计的 A 真相 → B 真相 → E 顶罪的框架。

- **保留“野生输入时间”的空白/模糊片段**：  
  - 让喜欢乱试时间的玩家，在任意阶段都能撞进一些抽象片段（比如“楼顶栏杆特写”“空教室”），增加个人体验的随机性，而不影响主线信息节奏。
