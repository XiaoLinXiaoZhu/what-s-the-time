# 脚本系统 V2 架构文档

## 概述

V2 架构使用绞杀藤模式（Strangler Fig Pattern）从 TS 脚本迁移到 MD 解析，核心改进是**支持 scope 循环跳转**。

## 架构对比

### V1 架构（旧）

```
TS 脚本文件 → ScriptSegment → Services → displayedLines
     │
     └── 选择分支通过 choices[].lines 嵌套实现
         └── 无法支持循环（会导致无限嵌套）
```

### V2 架构（新）

```
MD 文件 → script-compiler → ScriptSegmentV2[] → GameEngine → displayLines
   │                              │
   │                              └── 每个 scope 编译为独立 segment
   │                                  └── 支持循环跳转
   │
   └── ## scope 定义子片段
       └── - [选项](#scope) 锚点跳转
```

## 核心文件

| 文件 | 职责 |
|------|------|
| `src/types/script-v2.ts` | V2 类型定义 |
| `src/engine/GameEngine.ts` | 游戏引擎，管理状态和导航 |
| `src/parsers/script-compiler.ts` | MD → ScriptSegmentV2[] |
| `src/parsers/text-compiler.ts` | 文本 → TextNode[] |
| `src/components/GameViewV2.vue` | V2 游戏视图 |
| `src/components/v2/*.vue` | V2 行组件 |

## MD 剧本格式

```markdown
---
id: SEGMENT-ID
time: HH:MM | START | *
description: 片段描述
loop: P0 | P1 | *
unlockFlags: [FLAG1, FLAG2]
---

### narration
旁白文本

### dialogue 角色名
对话文本

### choice
- [选项A](#scope-a)
- [选项B](#OTHER-SEGMENT-ID)

## scope-a

### narration
scope-a 的内容
```

## 循环跳转原理

1. **编译阶段**：每个 `## scope` 编译为独立的 `ScriptSegmentV2`
   - ID 格式：`${parentId}-${scopeName}`
   - 例：`TEST-MD-LOOP-ask`, `TEST-MD-LOOP-weather`

2. **运行时**：选择后查找目标 segment 并追加其 lines
   ```typescript
   // GameEngine.handleChoice()
   const resolved = this.getSegment(`${parentId}-${targetId}`) || this.getSegment(targetId)
   this.insertSubSegmentLines(resolved, lineIndex)
   ```

3. **循环支持**：因为是追加而非嵌套，可以无限循环
   ```
   ask → weather → ask → time → ask → ...
   ```

## 开发模式

App.vue 右上角有 V1/V2 切换按钮：
- **V2 (MD)**: 使用新的 GameEngine + MD 解析
- **V1 (TS)**: 使用旧的 Services + TS 脚本

## 迁移路径

1. ✅ 基础设施：GameEngine, script-compiler, text-compiler
2. ✅ V2 组件：GameViewV2, v2/*.vue
3. ✅ 测试片段：TEST-MD-CHOICE, TEST-MD-JUMP, TEST-MD-LOOP
4. ⏳ 迁移旧片段：将 TS 片段转换为 MD 格式
5. ⏳ 清理：删除 V1 代码

## 文本格式支持

| 语法 | 效果 |
|------|------|
| `**文本**` | 粗体 |
| `==文本==` | 模糊（悬停显示） |
| `{red}文本{/red}` | 红色 |
| `{bold}文本{/bold}` | 粗体 |
| `{blur}文本{/blur}` | 模糊 |
| `{strike}文本{/strike}` | 删除线 |
| `{br}` | 换行 |
| `{delay:0.5}` | 延迟 0.5 秒 |
| `{typewriter}文本{/typewriter}` | 打字机效果 |
| `{systemTime}` | 显示系统时间 |
| `\|` 或 `/` | 延迟 0.25 秒 |
