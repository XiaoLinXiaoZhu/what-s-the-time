**总体定位**

- 这是一个给未来“写剧本的你自己/合作者”看的**剧情指导**，不是玩家文案，也不是流程图。
- 目标：帮创作者保持方向感——记住这条“鱼”究竟是什么、故事应该如何被“作为房间”搭建、最大的风险在哪里。

---

**一、故事本质与目标情绪**

- 核心不是“聪明的谜题”，而是：一个少年通过控制与叙事，把两段关系杀到只剩自己能承受的版本。
- 目标情绪：更接近“读完《人间失格》后的沉默与虚无”，而不是“解开谜题的爽感”或“正义审判的痛快”。
- 玩家体验的本质：不是破案，而是**一步步进入主角的精神结构**，直到意识到自己也在参与这场控制。

---

**二、真实时间线的角色：创作者的谜底，不是玩家的起点**

- 你已经有一条很完整的真实时间线（童年 → A 案 → A 案后 → B 案 → B 案后 → 多周目 → Meta）。
- 对创作者：真实时间线是**稳定的石头**，支撑一切，是你们的“工作底板”。
- 对玩家：真实时间线**不应该作为引导路线被端上去**。  
  玩家看到的不是“整理好的一条线”，而是一堆记忆碎片，这条线只在创作者脑中是完整的。

---

**三、从“线”到“房间”：内容组织方式**

- 丢掉“玩家体验线 = 一条导游路线”的想象。  
  换成：玩家被关在一座由**记忆房间**组成的建筑里，每个房间对应一个或一组时间点。
- 房间示例：  
  - `03:15`：童年黑房间，吵架声，尘土和恐惧的气味。  
  - `08:05`：早餐路上的阳光、三明治与被否决的包子。  
  - `10:37 (A)`：旧天台，风、松动栏杆，小石子的重量。  
  - `10:37 (B)`：新天台/新冲突，过曝的阳光，像褪色照片。  
  - `11:00 (A/B)`：问话室，两次近似场景的镜像。  
  - `18:40`：黄昏，血的气味，E 戴上手套的“喀嗒”。
- 创作时把注意力从“怎么带玩家走到结局”，转为“把这些房间造得尽可能有生命感和矛盾感”。

---

**四、每个房间需要的“标签”与写作重点**

为每个关键时间点（房间）准备至少四个维度的设计：

1. **谎言**：  
   - 这个房间在主角/社会的叙事里，呈现的“表面说法”是什么？  
   - 例：`10:37(A)` 的谎言是“意外”“救人失败”。

2. **真相**：  
   - 如果把所有遮掩拿掉，这个房间真正发生了什么？  
   - 例：预谋破坏栏杆、利用 H 的冲势，“顺势一推”。

3. **第一次进入时的感受/气味**：  
   - 玩家第一次看到这个片段时，应该闻到什么？是甜蜜、心疼、疑惑，还是温暖里的一根冷针？  
   - 写法上偏“直观体验”，让玩家自然站到“主角是受害者”的位置。

4. **在知道更多之后再回看的感受**：  
   - 文本本身可以不变，但要在一开始就埋下足够多的细节，使得当玩家掌握更多信息后，再回看同一段时，会自动重排意义。  
   - 例：  
     - `08:05` 三明治：第一次是体贴；后来变成控制的小笼子。  
     - 同样的一句“你胃不好”，前后含义完全不同。

> 重点：房间本身可以不“版本切换”，改变的是玩家的视角和知识量。  
> 强故事的过去**不需要被重写**，它只需要被反复照亮。

---

**五、可打散的“肉”与必须线性的“骨架”**

把整个故事拆成两层：

1. **可以非线性、自由探索的部分（Flesh）**

   - 与 A、B、主角、E、H、D 的日常相处片段；  
   - 各种侧面评价（别人眼中的主角、A、B）；  
   - 背景信息、校园氛围、小细节（饮食喜好、说话习惯等）；  
   - 大部分指向真相的线索碎片（但非“关键翻面”那一刀）。

   这些都适合做成散布在时间表上的房间，允许玩家以各种顺序进入。

2. **必须线性的情绪“脊柱”（Spine）**

   至少包括这几级“锁”：

   1. **B 的消失**：  
      - 从“日常糖衣”进入“有问题/不见了”的阶段。  
      - 触发条件可以是：玩家体验了足够多 B 的日常房间后，自动被推进。

   2. **“两个她”的发现**：  
      - 玩家意识到：A 与 B 并非一个人，但主角在记忆里混淆她们。  
      - 表现为某个细节对冲（如三明治/包子/饮料喜好、措辞差异）。

   3. **Meta 对峙**：  
      - 主角开始意识到玩家在“翻他的记忆”，并产生敌意或反问：“你到底想看什么？又是 10:37。”  
      - 这是从“调查他”到“和他对视”的转折。

   4. **最后的交接仪式**：  
      - 理想化的“她/C”出现，把停在 10:37 的手表交给玩家：“轮到你了。What’s the time?”  
      - 不可跳过，是故事的收束动作。

> 实践建议：  
> - 先在设计层面把这四个锁的**触发条件**想清楚（解锁多少房间、获得哪些信息、连续几次输入某些时间等）。  
> - 其它房间可以围绕这些锁任意铺设，但锁本身的顺序尽量不要被打乱。

**六、人物写作的重点提醒**

- **主角**：  
  - 不是简单“坏人”或“病娇”，而是用“受害者表演”求生长大的控制者。  
  - 他的魅力/危险在于：他对“理想她”的追求是真诚的，只是手段极端到毁灭别人。  
  - 玩家要在“他太可怕了”和“我有点懂他”的夹缝里难受。

- **A / B**：  
  - A = 理性反抗者，被暴力终结；B = 直觉反抗者，被替身化并毁灭。  
  - 重要一点：玩家在“糖衣阶段”的感情主要投向 B，真相揭示后，才意识到自己也在把 B 当替身——与主角共犯。

- **E**：  
  - 不是简单的“舔狗共犯”，而是：  
    - 被自己的沉默和无力感折磨了一年；  
    - 用“成为共犯”同时完成赎罪和自我实现（成为他生命中无法替代的那个人）。  
  - 在 Meta 层，她和理想化 C 有重叠，是对玩家说话的“她”的重要候选面孔。

- **H / D**：  
  - H：代表“被社会偏见轻易牺牲掉的人”；  
  - D：代表普通人的善意、友情的坚持与细节记忆，是少数真正站在 B 那一侧的人性锚点。

> 写作时的提醒：  
> - 永远先问：“这个片段里，谁在疼？疼什么？”  
> - 再想“它在结构上起什么作用”。不要反过来。

---

**七、关于多周目、时间输入与 Meta 的叙事意义**

- 时间输入 ≈ 玩家被迫用主角的方式回忆：  
  - 主角的记忆不是按“发生顺序”排的，而是按某些执念时间（10:37、11:00、08:05、03:15）聚合。  
  - 玩家在时间轴上乱戳，像是在操作一个精神病人的回忆播放器。

- 多周目 ≈ 主角对记忆的不断剪辑与重写尝试：  
  - 有的房间清晰、有的模糊、有的被强行“黑屏”。  
  - Meta 段：当玩家反复去同一个时间点，主角开始察觉到“被观看”，形成另一层对抗。

- 最终 Meta：  
  - 现实时间输入打破“这是游戏”的安全感，指向：  
    - 你在现实中也花了这么多时间，反复观看这段暴力；  
    - 你和主角一样，有一种“控制/理解”的冲动在推动你。

**八、工作方法建议（给未来写剧本时看的）**

1. **先造房间，不要先画路线**

   - 拿卡片/文档列出所有你确信需要的时间点房间；  
   - 为每个房间写好“谎言、真相、气味、第一次/第十次感受”；  
   - 先写 1～2 个房间的完整文本（建议 `08:05` 与 `18:40`），检验它们“是否活着”。

2. **等房间有了生命，再回头抽“锁”和“最小主干”**

   - 在房间之间找出自然的情绪转折点，确立锁；  
   - 只在锁上施加线性约束，其余交给玩家自由探索。

3. **防止项目死在“太聪明”上**

   - 每次你兴奋于一个结构/诡计设计时，强迫自己问一句：  
     - “这段里，具体谁在痛？我写清楚了吗？”  
   - 定期回去坐在 `03:15` 那个房间里，回想那个黑房间的气味。  
     - 这是整个故事的锚点，提醒你：这不是解谜玩具，而是一个关于控制与痛苦的故事。
