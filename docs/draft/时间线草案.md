  - 与 A 交往开始：普通校园恋爱，甜 + 稍微粘人。  
  - **P00-01：A 和主角的"正常早晨"**（08:05 闪回）：  
    - 纯甜日常，A 和主角一起走路上学，看起来只是普通情侣，埋一点主角微妙的控制欲（比如替她决定早餐）。  
  - **P00-02：A 第一次说"我们该谈一谈"**（12:10 闪回）：  
    - 教室午休，A 暗示"最近有点累""我们找个时间聊聊"，主角敷衍过去，为后面 08:20 约楼顶做铺垫。

  - 注意楼顶/高处结构，站位、栏杆、门锁开放时间；  
  - 观察 A 与 C 的接触频率，觉得“出轨”成立；  
  - 在心里排演“意外落下”的合理叙事（比如：摔倒、拉拽失败）。

#### 阶段 2：一年前 A 案当天（Day A）

> 这一天是第一次杀人的完整链条。

- **08:20 左右**（可作为二级时间，用于两问一锁）：  
  - **P01-01：A 发约楼顶的消息**（08:20 闪回）：  
    - 手机上只显示一条消息："等会儿上去聊聊。"+ 时间戳 08:20，是后来两问一锁的第二问答案之一。  
    - 主角表面答应，心里把这当作"机会"。

- **10:05（楼顶前的"平静几分钟"）**：  
  - **P01-02：10:05 楼顶前的"平静几分钟"**（10:05 闪回）：  
    - 两人刚到楼顶，聊着看似普通的话题，站位危险；主角看表，说"时间还早"，气氛微妙。

- **10:07（旧案·A 被杀）**  
  - 地点：校园楼顶。  
  - **P01-03：10:07 A 被推下（假闪回版）**（10:07 玩家第一次命中）：  
    - 模糊版楼顶记忆，只见踉跄和坠落，不见推的动作，主角自述"我没拉住她"。  
  - **P01-04：10:07 A 被推下（真相版）**（10:07 + 第二问 08:20）：  
    - 完整动作版——A 说"分手/秘密"，主角抓住手腕→有意识地"多推了一点点"，与假闪回形成对照。  
  - 事件：  
    - 二人争执升级，A 提出想结束关系 / 提及某个要揭露的秘密。  
    - 主角在"想拉住她"的伪装动作中发力，将她推下。  
  - 心理：  
    - 一瞬间的空白 → 迅速切换到"表演模式"。  
  - **P01-08：E 的远视目击**（16:10 纯闪回，E 视角）：  
    - E 坐在操场/走廊，从远处看到楼顶两个剪影，看到"不正常的动作"，然后选择低头装作没看见。

- **10:08–10:10（旧案·分钟切片，用于真相周目群）**  
  - **P01-05：10:08 A 死后 1 分钟，主角疯狂看表**（10:08 闪回）：  
    - 楼顶只剩"我"，连续多次看表，确认时间一步一步往后走但没倒回去，用看表次数写惊惧。  
  - 10:09：开始呼救、假装慌乱；  
  - 10:10：冷静下来，悄悄调整现场细节（比如把 A 掉落前的位置、脚印、手机摆放等弄成"争吵/意外"的样子）。

- **11:00（旧案·学校层问话）**  
  - 地点：教务处/办公室。  
  - **P01-06：11:00 学校第一次问话（旧案）**（11:00 第一次命中）：  
    - 教务处简单问话，主角说出"意外版本"，全程只用"她""那天"等代词。  
  - 事件：  
    - 老师、校方简单问话："当时发生了什么？"  
    - 初次版本的谎言成型：他讲述"她情绪失控、想往外走，我没拉住"。  
  - 备注：这个 11:00 对应 Day A 的"第一次陈述"。

- **12:30（旧案·警局初步笔录）**  
  - 地点：警局 / 校园内暂时设点。  
  - **P01-07：12:30 警局初步笔录（旧案）**（12:30 闪回）：  
    - 警官 G 做第一次正式笔录，主角第一次有意识地往替罪羊身上引导矛头。  
  - 事件：  
    - 主角用略微润色的版本再讲一遍，开始把矛头有意地带向 C。  

- **傍晚（18:xx）**  
  - 家里/寝室场景，主角在亲友面前用“悲伤版本”讲故事，强化“我=受害者”的社会形象。

#### 阶段 4：心理裂变与代偿人格形成（A 案后一段时间）

- 夜间独白、噩梦：  
  - 主角反复看到 10:07，但总在最关键画面"卡顿/雪花"。  
- 内心操作：  
  - 把 A 的"危险部分"剪掉，只保留温柔片段；  
  - 在脑内拼出一个理想化的"她"：不会背叛、不会离开。
- 时间感扭曲：  
  - 他反复停留在"那天上午大概十点"的心理时间上，外界时间继续流动。

#### 阶段 7：B 案当天（Day Bx）

> 第二次杀人，时间上有意围绕 10:07 形成"模式重复"。

- **上午某时（例如 08:xx）**：  
  - **P05-01：早上 08:xx，B 约"放学聊聊"**（08:20/08:30）：  
    - B 对主角说"今天放学/下午想和你谈谈"，为第二问答案（15:10 等）埋线。  
    - 约定地点（不一定是楼顶，也可以是偏僻角落）。  
    - 这可以成为两问一锁里"那天从 HH:MM 开始不对劲"的候选时间。

- **10:05（等待 B 的焦虑）**：  
  - **P05-02：10:05 等待 B 的焦虑（看表）**（10:05）：  
    - 主角在约定地点等 B，一直看表，每一次看表都让他想到一年前同样的时间。

- **10:07（新案·B 被杀）**  
  - 可能也是在高处/隐蔽场所（与 A 案形成镜像）。  
  - **P05-03：10:07 杀 B（真相版）**（10:07 + 第二问 15:10 等）：  
    - B 说"我不是她"，主角重复 A 案的暴力模式，将她推下/动手，构成"模式重复"。  
  - 事件：  
    - B 试图划清界限 / 质问主角一些行为（谎言、偷窥、控制欲）：  
      - "你到底当我是谁？"  
      - "我不是你心里那个她。"  
    - 主角在"你也要离开我"的恐惧中，复刻当年动作，把她推下/实施暴力。  
  - 心理：  
    - 更快地进入"模板模式"，直接往"意外/她自己失控"的故事上套。  
  - **P05-05：E 清晰目击 B 案**（16:20 单独闪回，E 视角）：  
    - E 从楼下/对面看到整个动作，这次她完全确认"不是意外"，却仍然选择沉默到晚上。

- **不久后（10:08–10:xx）**：  
  - **P05-04：10:08–10:xx B 案后几分钟**（10:08/10:09/10:10 后期分钟切片）：  
    - 地点更隐蔽，无即时目击；主角短暂慌乱后快速进入"处理模式"，呼应 A 案但也有差异（没人会冲上来）。  
  - 与 A 案不同之处：  
    - 地点足够隐蔽 / 暂时无人目击；  
    - 主角有时间进行后续处理，而不是立刻被发现。  
  - **重要差异**：  
    - E 这一次并非模糊目击，而是清楚地看到主角杀人的过程（可以是从楼下/对面窗口等角度看到关键动作）。  
    - 她当下没有报警，而是陷入剧烈的震荡与否认："他真的做了这件事吗？我是不是又看错了？"

- **18:39（分尸前一分钟看表）**：  
  - **P05-06：18:39 分尸前一分钟看表**（18:39）：  
    - 垃圾站前，主角拖着袋子，一次次看表，文字用"还有一分钟"强调心理准备。

- **18:40（新案·分尸与垃圾袋 + 锯子，E 成为共犯）**  
  - 地点：偏僻的垃圾站/工地/楼下角落。  
  - 事件：  
    - 主角使用锯子等工具准备分尸，将尸块装入垃圾袋，打算逐步处理。  
    - E 出现在现场（可以是事先找到他，或一路跟随而来）。  
    - 在这场对峙中，E 做出了极端选择：  
      - 她没有指责/报警，而是对主角说类似的话：  
        - "她说，叫她 C。"  
        - "她会帮他处理掉 C。"  
      - E 提出自己来帮忙"处理掉 C"：  
        - 这里的 C 是一个高度压缩的符号：  
          - 现实层面：指现在这具 B 的尸体；  
          - 记忆层面：也指一年前死去的 A —— 那个一直缠着主角的"同名者"；  
          - 象征层面：指围绕这个名字的一切罪与纠缠。  
    - 行动上，E 不是旁观者，而是实际参与者：  
      - 她可能亲手提起垃圾袋、递上锯子，甚至直接动手分尸；  
      - 她以这种方式，把自己从"最后一个目击者"变成"现案共犯"。

  - **P05-07：18:40 分尸真相·E 成为共犯**（18:40 后期真相版）：  
    - E 出现，说"叫她 C""她会帮他处理掉 C"，戴手套、帮忙处理尸体，把 A/B/名字/罪全部压缩成一个 C。  
  - 文本层（前后期差异）：  
    - 前期周目：这一时间点被主角用"农活/兼职/家里让帮忙"解释，E 的存在被刻意遗忘，玩家只看到"垃圾袋 + 锯子"的日常伪装。  
    - 后期周目：同一 18:40 场景从另一视角重演，E 出现在画面中，说出"叫她 C""处理掉 C"之类的关键台词，真相版分尸过程被展示。

- **18:55（分尸后洗净的日常动作）**：  
  - **P05-08：18:55 分尸后洗净的日常动作**（18:55）：  
    - 主角或 E 在水龙头下洗手/洗锯子，只用"今天的水有点凉"这类话，把极度异常包成平常动作。

- **之后几天**：  
  - B 没来上课，D 最先感到不对劲（消息不回、电话关机）。  
  - 老师暂时用"请假/家里有事"安抚大家。
