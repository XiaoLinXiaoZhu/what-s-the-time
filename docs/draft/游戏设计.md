## 《What's the time》剧本大纲

### 一、游戏概述

- **类型**：多周目文字冒险，外壳为轻松的校园恋爱喜剧，内核是预谋杀人、记忆扭曲与叙事诡计。
- **核心机制**：每个周目开头都会出现类似的自问——「时间是……？」「不，是他在问我。」玩家自由输入时间（HH:MM），游戏据此播放对应的“记忆片段”。
- **玩家体验目标**：
  - 在前中段，玩家始终认为自己操控的是一个“被命运折磨的受害者”。
  - 在末段，通过真相周目和 meta 突破，意识到“我”是两起同名受害者案件的真凶，而玩家本身也在不断重播这场暴力。

---

### 二、核心真相（玩家不可见）

- **两次杀人，同名受害者**：
  - 一年前：主角在校园楼顶，于 **10:07** 有计划地杀死恋人 A（下文记为 A，同名 X）。
  - 现在：主角又将班上一位与 A 同名/同音的女生 B（同为 X）视作替代品，在类似心理驱动下再次杀死她，同样围绕 10:07，当下阶段选择分尸与抛尸。
  - 校园视角：
    - 旧案 A 已经“尘埃落定”，社会认定替罪羊 C 为凶手（或主要责任人）。
    - 新案 B 目前只被视为“失踪/请假”，没有明确死亡结论。

- **栽赃与自我受害叙事**：
  - A 案中，主角精心准备现场与证据，引导舆论与警方视线，将责任栽赃给“疑似出轨对象” C。
  - B 案中，主角潜意识复用同一套“故事模板”：同样的时间、类似的场景、相似的说辞。
  - 在自我叙事中，主角持续塑造“我只是受害者”“真正的怪物是别人”的形象，逐渐也相信了自己的谎言。

- **代偿人格与“她”的模糊身份**：
  - 杀死 A 之后，主角在内心形成了一个理想化的“她”：不会背叛、永远理解他，是被美化的 A 与罪恶感混合而成的投射。
  - 一年后遇到 B（与 A 同名），主角逐渐把这个理想化人格强行投射在 B 身上，对现实中的 B 性格视而不见。
  - 全游戏中，“她”这个代词长期不区分是 A 还是 B，直到真相周目才被一次性拆分。

- **时间与记忆的扭曲**：
  - 对主角而言，真正卡住他的，是第一次杀人那一刻的 **10:07**。
  - 多周目是他反复重播、剪辑这两起案件相关记忆的过程：有时是旧案、有时是新案、有时是幻想润色后的版本。
  - 玩家输入时间，其实是在操纵“我”愿意面对的记忆片段，而非上帝视角的真实纪录。

- **最终 meta 高潮**：
  - 在最后一个关键周目，“她”第一次明显打破第四面墙，要求玩家输入现实中的“现在时间”。
  - 若玩家输入错误，她报出系统当前时间：“不对，现在是 XX:XX，对吧？”
  - 这一步将“叙事时间 / 游戏时间 / 现实时间”折叠到一起，把玩家也拖入“你在重播这场暴力”的责任之中。

### 三、真实时间线概要（按现实顺序）

> 注：真实时间线只在创作与推理设计时使用，游戏中不会直接标明日期，只通过时间与场景暗示。

#### 1. 案发前（A 案）

- 主角与 A 交往，一开始是正常的校园恋爱。
- 随着时间推移，A 对主角的控制欲与不稳定感到不安，关系出现裂痕。
- A 掌握了主角的一些“致命秘密”（可后续细化，如作弊、偷拍、暴力事件等），主角感到被威胁。
- 主角开始酝酿杀意，计划在楼顶制造一场“意外”。

#### 2. 一年前，A 案当日

- A 与主角约在校园楼顶见面，表面上是“好好谈谈”。
- **10:07（旧案）**：争执到达临界点，主角在伪装拉扯/安抚的姿势下将 A 推下。
- A 死亡，主角短暂停顿后，开始伪装慌乱，呼救、演戏。
- 主角刻意引导证据与目击信息，指向与 A 关系暧昧的男生 C。

#### 3. A 案后续

- 警方介入调查，媒体渲染为“校园情杀/意外悲剧”。
- C 因既有的性格问题/前科/与 A 的关系，被迅速锁定为“最疑似”的一方。
- 官方结论倾向于：C 负主要责任或“意外事故下的加害者”，案件结案。
- 主角作为“死者恋人”被同情、关照，其怪异行为被解释为创伤后遗症。
- 在此过程中，他在问话与笔录中反复讲述同一套修饰后的故事，逐渐把自己也洗成“受害者”。

#### 4. 心理裂变与代偿人格

- 夜晚独处时，主角不断回放 10:07 的画面，却有选择地抹去自己的杀意与动作。
- 为了维持对 A 的爱与自我无罪的幻觉，他在心中创造出一个理想化的“她”，既像 A 又比 A “乖巧安全”。
- 真实的 10:07 被切碎成零散、模糊的片段，一部分记忆被压抑。

#### 5. 一年后，B 的出现（现在线开始）

- 新学期或转班后，出现了与 A 同名/同音的女生 B。
- 主角被名字强烈触发，将理想中的“她”影子投射在 B 身上，逐步忽视 B 的真实性格。
- 校园里不时有人以“那件事”“她当年”为话题闲聊，旧案残影与现实生活交织。

#### 6. B 案当日

- 在一系列误会、纠缠或情感冲突之后，主角再次陷入“她要离开我 / 她会毁掉我”的恐惧模式。
- 主角把当年的“成功模式”在心理中内化，并在某个时刻再次围绕 **10:07** 实施暴力，将 B 杀死。
- 这一次，为避免尸体迅速暴露，他选择分尸，将尸块装入垃圾袋，并使用锯子等工具处理。
- 当前校园层面，只认为 B “突然没来上课/回老家/转学”，官方尚未明确“失踪=死亡”。

#### 7. B 案后续（当前时间）

- 老师/警察/同学开始注意到 B 的长期缺席，出于惯例或职业敏感，启动“了解情况”。
- 主角因为“最后见到 B 的人之一 + 一年前旧案当事人”再次被关注。
- 警局/办公室的问话中，旧案与新案在语言层面被缝在一起——他们问“那时候”“她”，但不说清是哪一次哪一个“她”，这也是游戏叙事的主要模糊源。

### 四、核心玩法与时间输入机制

- **自由输入 + 有限锚点**：
  - 玩家在每个周目开头会看到类似台词：  
    - 「时间是……？」  
    - 「不，是他在问我。」
  - 玩家可以输入任意 HH:MM，系统将其解析为时间，并与内部的“故事时间表”匹配。
  - 仅有一部分时间点有对应剧情锚点，其余时间统一给出“记忆空白/模糊区间”的演出。

- **普通时间点（一问一答）**：
  - 如 08:05、18:40、23:15 等：  
    - 玩家输入命中锚点 → 直接播放对应场景。  
    - 未命中 → 进入统一的记忆空白（黑屏+心声）或模糊画面（例如楼顶栏杆、空教室等）。

- **重叠时间点（两问一锁）**：
  - 对于承载多个事件的“重叠时间”（如 10:07、11:00），采用两步锁定：
    1. 第一问：输入“现在几点？” → 玩家输入 10:07 / 11:00。
    2. 第二问：出现补问，例如  
       - 「我记得，那天是从几点开始不对劲的？」  
       - 玩家再输入一个时间（如约见时间/被通知时间），形成组合键 (10:07, 08:20) / (10:07, 15:10)。
  - 不同组合对应不同日期与事件：
    - (10:07, X₁) → 一年前 A 被杀的那天的 10:07。  
    - (10:07, X₂) → 现在 B 被杀的那天的 10:07。  
    - 同理，(11:00, Y₁)/(11:00, Y₂) 可指向旧案/新案的不同问话场景。

- **记忆空白与模糊区间**：
  - 对部分时间段（尤其是尚未解锁的关键区间，如 10:0x），可提供抽象、不完整的场景与台词：“大概是十点多……我不想想得太清楚。”
  - 对完全无关的时间（如深夜随机时间）则直接给空白：“那段时间对我来说，就是一片黑。”

### 五、主要角色与关系网

> 这一节用于创作与设计，游戏中通过对话/事件逐步显露，不直接用“设定集”的口吻出现。

#### 1. “我”（主角）

- **表层印象**：
  - 性格内向、略显木讷的学生，成绩中上。
  - 在旁人眼中，是“一年前那件事中最可怜的人”，被同情，被特殊照顾。
- **真实面貌**：
  - 两起案件的实施者：  
    - 对 A 是有计划的预谋杀人 + 栽赃；  
    - 对 B 是模式化重复犯罪 + 分尸抛尸。
  - 强烈的控制欲、不安全感与自我受害叙事交织。
- **语言与代词特点**：
  - 时常用“他”指代“当年的那个我”，以此切割责任。
  - 对 A/B 几乎都只用“她”“那个人”，拒绝正面说出全名。
- **核心冲突**：
  - 一边坚持自己是“被事件选中的受害者”，一边不断复刻加害行为。
  - 被时间（10:07）和同名“她”困住，拒绝前进。

#### 2. A（旧案受害者，女，同名 X）

- **表层印象（他人口中）**：
  - 优等生、独立，有些人觉得她“太有主见/太强势”。
  - 与 C 有亲近关系（真实情况可暧昧或仅是好友）。
- **真实关系**：
  - 知道主角的一些“脏事”（具体可在后续周目中逐步揭示），对他构成实质威胁。
  - 尝试与主角拉开距离或提出分手/改变关系。
- **在游戏中的呈现**：
  - 长时间只以“她”“那时的她”出现，少量闪回。
  - 名字与 B 相同或极近，使玩家一度完全混淆。

#### 3. B（现案受害者，女，同名 X）

- **表层印象**：
  - 与 A 同名/同音，在现实性格上却明显不同（例如更随性/更阳光/更直率）。
  - 主角的同班或邻班同学，与他有普通但逐渐升温的互动。
  - 起初真心把主角当"需要照顾的可怜人"，有一段短暂的好感期。

- **真实内心与冲突**：
  - 自己有明确的人生目标（考学/离开小镇/追求某种生活），对被他人"束缚"非常敏感。
  - 对主角的好感起点是同情与共鸣，而不是简单的恋爱心动。
  - 随着时间推移，她越来越清楚地意识到：
    - 主角看她的眼神里有"别人"的影子；
    - 很多对她说的话，其实是在对那个已经死去的 A / 理想化"她"说。

- **弧光（前中后）**：
  - 前期：  
    - 轻松日常线女主，与主角一起上学、吃饭、聊天，会开玩笑说"你又想到她了？"  
    - 玩家容易把她当作"攻略对象"。
  - 中期：  
    - 开始在对话中试探："你是不是把我当成谁了？"  
    - 对 D 倾诉"他有时候看我的眼神很奇怪"，但仍不想伤害他。
  - 后期（B 案当天前后）：  
    - 选择约主角"好好谈谈"，想明确拒绝/划清界限："我不是你心里那个她。"  
    - 这次谈话成为第二次杀人的直接导火索。

- **叙事功能**：
  - 作为 A 的"活替身"存在，被主角暴力地套进"角色 C"的模具。
  - 她的真实人格通过 D 的视角慢慢补完，让玩家意识到：她不是模板，而是被误读后被杀的人。

#### 4. 替罪羊（男，一年前旧案的背锅者）

- **表层印象**：
  - 与 A 有暧昧/密切关系，有"坏名声"（暴躁、打架、逃课等标签）。
  - 在一年前的舆论中被迅速判定为"最可疑的人""根本不正常"。

- **真实情况**：
  - 被主角有意识地栽赃：主角利用他与 A 的关系和既有偏见，将矛头导向他。
  - 参与程度可能极低，甚至完全无辜，只是"长得像怪物的人"。
  - 最终在社会叙事中被固定为"那件事的 C 位"，为主角腾出了"我=受害者"的位置。

- **在游戏中的呈现**：
  - 多通过他人口中出现，很少正面出场（可以只有一两次远景或背影）。
  - 他的存在让玩家在中期一度相信"也许真的是他干的"，直到后期细节反转。

#### 5. D（B 的朋友/同学，目击"最后一次"）

- **身份**：
  - B 的好友 / 室友 / 社团同伴，对 B 的日常状态最敏感的人。
  - 与主角有普通同学关系，前期对他并无敌意。

- **关键信息**：
  - "最后一次见到 B 和主角在一起"的见证人。
  - 在现在线中，一旦案件曝光，D 是最早会公开指出主角问题的人之一。

- **情感与立场**：
  - 核心驱动力是"为 B 说话"，而不是"破案"：  
    - 大家都说"她可能只是有事回家"，只有 D 坚定认为"她不会不打招呼就消失"。
  - 可设定为：  
    - 要么暗恋 B，对主角有潜在敌意；  
    - 要么将 B 当作最重要的朋友，对任何伤害她的人零容忍。

- **弧光**：
  - 前期：  
    - 调侃主角和 B 像情侣，帮他们打圆场，也会说"他其实还不错啦"。  
  - 中期：  
    - 逐渐收集矛盾信息：B 的不安 + 主角奇怪的反应 + 老师/警察的问话。  
    - 对主角态度从中立 → 怀疑。  
  - 后期：  
    - 明确质问主角："那天你到底对她说了什么？"  
    - 对玩家/主角说出类似："如果这次也是因为你，我永远不会原谅你。"

- **叙事作用**：
  - 代表"身边人"的视角，让玩家看到一个不再相信主角叙事的人。
  - 在多周目中，他的台词从模糊到笃定，是主角"受害者形象"被瓦解的关键力量。

#### 6. E（一年前的目击者 + 现案共犯，试图成为现实中的 C）

- **身份**：
  - 一年前就与主角在同一所学校的同学/学长/学姐，现在仍在校园或附近活动。
  - 长期在心理上"站在主角这边"，想成为那个"真正理解他的人"。

- **一年前的秘密（A 案）**：
  - 在 A 被杀的那天，E 曾远远看到楼顶上一瞬间异常的画面：  
    - 例如主角明显的推搡动作，或 A 掉下去前主角古怪的表情。
  - 当时她无法确认，也不敢说，事后长期用"可能是错觉"说服自己，形成强烈负罪感：  
    - "如果我当时说出来，也许就不会有现在这一切。"

- **现在的选择（B 案，共犯）**：
  - 在 B 案当天，E 这一次不是模糊目击，而是**清楚地看到主角杀人的过程**。
  - 她没有报警，而是在晚上 18:40 左右，出现在主角准备分尸/处理尸体的现场。  
  - 在那一幕，她做出极端选择：
    - 对主角说类似："她说，叫她 C。"  
    - "她会帮他处理掉 C。"
  - 这里的 **C 是高度压缩的符号**：
    - 既指旧案中的 A（那个始终缠绕主角的名字与记忆）；  
    - 也指现在 B 的尸体（这具同名者）；  
    - 更指围绕这个名字的一切罪和纠缠。
  - E 的潜台词是："让我来帮你把这一整个 C 处理掉——过去和现在，都交给我。"

- **与主角的情感关系**：
  - 她的爱里有真实的心动，也有逃避：  
    - 不愿承认自己喜欢的是"杀人犯"，所以选择加入犯罪，把自己也变成"同一种人"。  
  - 她想成为主角生命中独一无二的"共犯"和"支撑点"，所以主动抢占"C 这个位置"。

- **弧光**：
  - 前期：温柔的救赎线，鼓励主角"放下那件事"，带他去别的地方走走。  
  - 中期：在看到主角对 B 的态度后，产生强烈既视感，但继续自我欺骗。  
  - 后期（B 案当天及之后）：  
    - 从目击者升级为共犯，与主角一起处理尸体，自称 C；  
    - 在她的逻辑里，这是对一年前沉默的"补偿"，也是一种扭曲的"自我牺牲"。

- **叙事作用**：
  - 让故事从"一个人的心理问题"升级为"多人共同成就的怪物系统"：  
    - 一年前的沉默 + 现在的帮凶。  
  - 在终盘，她既是控诉者，也是同犯者：  
    - "如果我当年说出来，就不会有第二个她；  
      可是现在，我也变成了怪物的一部分。"

#### 7. F（班主任 / 导师）

- **身份与姿态**：
  - 负责主角班级/课题，肩负“教育者”的位置。
  - 一开始对主角极其同情，帮他挡掉很多麻烦。
- **变化轨迹**：
  - 前期：把主角所有怪异行为都解释为创伤后遗症。  
  - 中期：隐隐不安，开始注意到他讲述中的矛盾。  
  - 后期：面对警方与学校压力，摇摆在“保护学生”和“追求真相”之间。
- **叙事作用**：
  - 代表“制度/成年人”的视角，体现社会对主角态度的逐步变化。

#### 8. G（警官）

- **身份**：
  - 一年前旧案、现在失踪案都参与过的警官，一直记得主角这张脸。
- **风格**：
  - 不夸张、不戏剧化，语气平静但有压迫感。
- **叙事作用**：
  - 在问话室中使用大量模糊代词："那件事""她""那时候"，让旧案与新案在语言层面缝合。
  - 对玩家而言，是"时间与法律"的代表人物：你可以躲开记忆，但躲不开他。

#### 9. 角色 C（概念性的"位置"）

- **定义**：
  - C 不再是某个具体人的名字，而是一个"角色位"：  
    - 那个主角心里"可以为我做任何事的人"；  
    - 那个被社会故事认定为"那件事中的怪物/关键人物"的位置。

- **占据过 / 试图占据 C 位的人**：
  - 旧案中的替罪羊：被社会故事塞进"怪物"的 C 位。  
  - A：在主角心中的最初"理想恋人 C"。  
  - B：被主角暴力地当成新的 C（替代 A 的位置），因此丧命。  
  - E：在 B 案后主动说"叫她 C"，试图成为现实中"为你背负一切的人"。

- **叙事意义**：
  - 强化"名字/位置可以被替换，但暴力模式一直在"的主题。  
  - 让玩家在多周目中不断问："现在被当作 C 的，是谁？谁在试图抢这个位置？谁被扔进这个位置然后被丢弃？"

### 七、叙事诡计与主题提示（写作时的注意点）

- **代词模糊**：尽量用“他/她/那个人/那件事”，延后真名的出现。
- **时间不标日期**：只给出“几点几分”，让不同年份的同一时间在玩家认知中重叠。
- **同名混淆**：A 与 B 同名/同音，E 想成为新的“她”，共同构成“谁配得上这个名字”的暗线。
- **多周目记忆残留**：后期“她”可以偶尔提及玩家此前输入的时间，但不要提前表现出强烈的系统感，留力给最终 meta。
- **现实时间突破**：严格只在最后一次使用系统时间纠正玩家输入，保持冲击力。
