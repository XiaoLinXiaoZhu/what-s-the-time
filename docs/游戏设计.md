## 《What's the time》剧本草案

### 一、游戏概述

- **类型**：多周目文字冒险，外壳为轻松的校园恋爱喜剧，内核是预谋杀人、记忆扭曲与叙事诡计。
- **核心机制**：每个周目开头都会出现类似的自问——「时间是……？」「不，是他在问我。」玩家自由输入时间（HH:MM），游戏据此播放对应的“记忆片段”。
- **项目核心愿景：**
打造一部文学性的、拒绝提供简单道德答案的心理惊悚文字冒险游戏。其最终体验并非解谜的快感或情感的宣泄，而是类似于读完《人间失格》后，那种对人性深渊的凝视所带来的、无法言说且挥之不去的压抑与虚无感。
- **最终形态的轮廓：**
    *   **前期**：拖长日常甜蜜情侣部分，减慢异常的导入。用B的生日作为时间锚点，包裹主角和D为其准备生日礼物等日常故事。将主角和玩家绑定在一起，让玩家代入主角体验类似于校园恋爱题材的作品。玩家与主角的情感锚点是**和B的情感**，但这种情感从一开始就带有病态控制的色彩——玩家代入的是一个看似深情、实则正在将恋人"物品化"的角色的视角。
    *   **中期**：玩家与主角的情感锚点是**"B怎么死了？我是受害者"**这样的共情路径。玩家需要扮演主角证明自己的无罪和伤痛。
    *   **中后期**：玩家逐渐发现主角的异常，跟随这个异常重新探索新的时间片段以及整理整个故事的真实脉络。玩家的情绪从"我爱的人怎么死了？"转变为"**我们（我和主角）的爱，是不是从一开始就错了？**"这种自我怀疑，这种对自己前期情感投入的否定，是通往"人间失格"式虚无感的入口。
    *   **后期**：meta元素让玩家扮演主角执行杀人，和C（理想化的"她"，或可理解为游戏最终的E）跳出游戏框架，询问主角时间。此时玩家的现在时间和游戏的虚拟时间达成统一。

### 二、核心真相（玩家不可见）

- **两次杀人，同名受害者**：
  - 一年前：主角在校园楼顶，于 **10:37** 有计划地杀死恋人 A（下文记为 A，同名 X）。
  - 现在：主角又将班上一位与 A 同名/同音的女生 B（同为 X）视作替代品，在类似心理驱动下再次杀死她，同样围绕 10:37，当下阶段选择分尸与抛尸。
  - 校园视角：
    - 旧案 A 已经“尘埃落定”，社会认定替罪羊 C 为凶手（或主要责任人）。
    - 新案 B 目前只被视为“失踪/请假”，没有明确死亡结论。

- **栽赃与自我受害叙事**：
  - A 案中，主角精心准备现场与证据，引导舆论与警方视线，将责任栽赃给“疑似出轨对象” C。
  - B 案中，主角潜意识复用同一套“故事模板”：同样的时间、类似的场景、相似的说辞。
  - 在自我叙事中，主角持续塑造“我只是受害者”“真正的怪物是别人”的形象，逐渐也相信了自己的谎言。

- **代偿人格与“她”的模糊身份**：
  - 杀死 A 之后，主角在内心形成了一个理想化的“她”：不会背叛、永远理解他，是被美化的 A 与罪恶感混合而成的投射。
  - 一年后遇到 B（与 A 同名），主角逐渐把这个理想化人格强行投射在 B 身上，对现实中的 B 性格视而不见。
  - 全游戏中，“她”这个代词长期不区分是 A 还是 B，直到真相周目才被一次性拆分。

- **时间与记忆的扭曲**：
  - 对主角而言，真正卡住他的，是第一次杀人那一刻的 **10:37**。
  - 多周目是他反复重播、剪辑这两起案件相关记忆的过程：有时是旧案、有时是新案、有时是幻想润色后的版本。
  - 玩家输入时间，其实是在操纵“我”愿意面对的记忆片段，而非上帝视角的真实纪录。

- **最终 meta 高潮**：
  - 在最后一个关键周目，"她"第一次明显打破第四面墙，要求玩家输入现实中的"现在时间"。
  - 若玩家输入错误，她报出系统当前时间："不对，现在是 XX:XX，对吧？"
  - 这一步将"叙事时间 / 游戏时间 / 现实时间"折叠到一起，把玩家也拖入"你在重播这场暴力"的责任之中。
  - **最终结局设计**：
    *   所有真相揭晓后，与玩家对话的，不再是A或B的亡魂，也不是E，而是那个被主角在内心创造出来的、完美的、理想化的**"她"（C）**。这个C，是主角一切罪行的根源和最终目的。在最终meta环节，E与这个理想化的"她"（C）融合，成为跳出游戏框架与玩家对话的存在。
    *   C的声音平静而诱人，她对玩家说："你看，他失败了。他不够纯粹，不够坚定，所以他弄脏了一切。但他追求我的心，是真的。那种想要留住永恒、拒绝任何瑕疵的渴望……你，屏幕前的你。你在花费了数小时，看遍了他所有的失败、丑陋和挣扎之后……你难道没有一瞬间，理解他吗？理解那种，想要创造一个'绝对完美的世界'的冲动吗？……手表还给你。现在，轮到你了。What's the time?"
    *   游戏界面回到最初的输入框，等待玩家输入一个新的`HH:MM`。但这一次，玩家知道，他输入的不再是"过去的时间"，而是**"他自己的、新的'10:37'"**。游戏不会结束，它变成了一个开放的循环。玩家可以关闭游戏，但那个问题会永远留下。
    *   **结局内核**：
      - 它不审判玩家的"观看行为"，而是拷问玩家内心深处是否也存在与主角相似的**"控制欲"和"完美主义的魔鬼"**。
      - 它不提供任何答案或解脱。AB没有安息，主角和E的未来是虚无，而玩家则被赋予了一个可怕的"权力"——成为下一个悲剧的开端。
      - 它将"人间失格"的体验，从观察主角，变成了反思自我。玩家在凝视深渊之后，发现深渊也在凝视自己，甚至向自己递出了一块怀表。

### 三、核心人物和叙事设计

1.  **主角 ("我")**：他的行为逻辑根植于其童年创伤中"要么控制，要么被控制"的生存法则。他对A和B的"爱"，本质是一种将其"物品化"并纳入自己绝对控制范围的欲望。他与玩家的对抗，是他维护内心秩序的最后挣扎。

2.  **A与B (受害者)**：她们的区分度在于**"反抗方式"**。A是理性的、有计划的"谈判式"反抗；B是感性的、直觉的"质问式"反抗。她们的悲剧不仅在于被杀，更在于她们鲜活的个性被主角用同一个名为"X"的模具所覆盖和抹杀。

3.  **E (共犯)**：她的动机被深化为一种**扭曲的自我实现**。她将参与犯罪视为摆脱平庸、与心爱之人建立终极"共犯"关系的仪式。她主动占据"C"位，是一种将自己献祭给这段病态关系的极端行为。在最终meta环节，E与主角内心创造出的理想化"她"（C）融合，成为跳出游戏框架与玩家对话的存在。

4.  **H与D (外部视角)**：H是"社会偏见"的牺牲品，其存在证明了主角操控叙事的能力。D则是"日常直觉"的代表，她的怀疑来自最朴素的友情和观察，为冰冷的悬疑故事提供了宝贵的人性温度。

5.  **叙事诡计的实现方式（纯文本环境）**：
    *   **甜蜜中的诡异**：前期日常不再是单纯发糖，而是充满了"修正对方品味"、"替对方做决定"等**病态控制的细节**，为后期反转提供坚实的铺垫。这些细节在当时看来可能只是"占有欲强"或"有点怪"，但事后回想全是致命的警报。
      *   **示例1：礼物事件**：主角和D一起为B准备生日礼物。主角否决了D的所有提议，理由是"那些都不适合她"。他最终选了一件礼物，不是因为B喜欢，而是因为这件礼物**符合"他心中理想的B"应该有的样子**。D可能会吐槽一句："这真的是B会喜欢的东西吗？" 主角则会温柔而坚定地回答："她会的。她只是自己还不知道。"这不再是单纯的发糖，而是一颗裹着糖衣的毒药。玩家在享受甜蜜的同时，隐约会感到一丝不协调和控制感。
      *   **示例2：对话中的"覆盖"**：B分享她喜欢的乐队，主角听完后说："我明白了。但我觉得，你听这首古典乐时，会更美。" 他不是在推荐，而是在**"修正"B的品味**。爱在这里变成了一种模具，主角在试图把活生生的B塞进他理想的模具里。
    *   **文本形式的"故障"**：通过文本的划线修正（`~~红色的~~蓝色的`）、伪代码报错（`[ERR]`）、人称的瞬间漂移（`我` -> `你`）等形式，来暗示记忆的不可靠性和Meta层面的存在。

6.  **玩家情感锚点的设计（四个阶段）**：
    *   **前期**：拖长日常甜蜜情侣部分，减慢异常的导入。用B的生日作为时间锚点，包裹主角和D为其准备生日礼物等日常故事。将主角和玩家绑定在一起，让玩家代入主角体验类似于校园恋爱题材的作品。玩家与主角的情感锚点是**和B的情感**，但这种情感从一开始就带有病态控制的色彩——玩家代入的是一个看似深情、实则正在将恋人"物品化"的角色的视角。
    *   **中期**：玩家与主角的情感锚点是**"B怎么死了？我是受害者"**这样的共情路径。玩家需要扮演主角证明自己的无罪和伤痛。
    *   **中后期**：玩家逐渐发现主角的异常，跟随这个异常重新探索新的时间片段以及整理整个故事的真实脉络。玩家的情绪从"我爱的人怎么死了？"转变为"**我们（我和主角）的爱，是不是从一开始就错了？**"这种自我怀疑，这种对自己前期情感投入的否定，是通往"人间失格"式虚无感的入口。
    *   **后期**：meta元素让玩家扮演主角执行杀人，和C（理想化的"她"，或可理解为游戏最终的E）跳出游戏框架，询问主角时间。此时玩家的现在时间和游戏的虚拟时间达成统一。

### 四、核心玩法与时间输入机制

- **自由输入 + 有限锚点**：
  - 玩家在每个周目开头会看到类似台词：  
    - 「时间是……？」  
    - 「不，是他在问我。」
  - 玩家可以输入任意 HH:MM，系统将其解析为时间，并与内部的"故事时间表"匹配。
  - 仅有一部分时间点有对应剧情锚点，其余时间统一给出"记忆空白/模糊区间"的演出。

- **阶段和flag系统**：
  - 游戏通过阶段和flag系统，避免玩家在前期直接跳转到后期的时间点上。
  - 穷举行为在预期中，也有其意义：玩家随意探索时间，可能拿到一些线索，但阶段和flag框定了一部分内容，穷举也能够拿到有效信息，进而继续探索。
  - 整个游戏最大的魅力就是**"下一个时间点在哪里"**，这与游戏名称"What's the time"紧密相关。

- **普通时间点（一问一答）**：
  - 如 08:05、18:40、23:15 等：  
    - 玩家输入命中锚点 → 直接播放对应场景。  
    - 未命中 → 进入统一的记忆空白（黑屏+心声）或模糊画面（例如楼顶栏杆、空教室等）。

- **重叠时间点（两问一锁）**：
  - 对于承载多个事件的"重叠时间"（如 10:37、11:00），采用两步锁定：
    1. 第一问：输入"现在几点？" → 玩家输入 10:37 / 11:00。
    2. 第二问：出现补问，例如  
       - 「我记得，那天是从几点开始不对劲的？」  
       - 玩家再输入一个时间（如约见时间/被通知时间），形成组合键 (10:37, 08:20) / (10:37, 15:10)。
  - 不同组合对应不同日期与事件：
    - (10:37, X₁) → 一年前 A 被杀的那天的 10:37。  
    - (10:37, X₂) → 现在 B 被杀的那天的 10:37。  
    - 同理，(11:00, Y₁)/(11:00, Y₂) 可指向旧案/新案的不同问话场景。
  - 需要回答两个时间点才展开相关剧情的设计已经足够，且游戏中还做了简单的选择和分支系统，玩家可以决定主角的行为，从而创造小的走向上的不同，也能消除一定重复感。

- **记忆空白与模糊区间**：
  - 对部分时间段（尤其是尚未解锁的关键区间，如 10:0x），可提供抽象、不完整的场景与台词："大概是十点多……我不想想得太清楚。"
  - 对完全无关的时间（如深夜随机时间）则直接给空白："那段时间对我来说，就是一片黑。"
  - **对"错误"输入的反馈**：当玩家输入一个不存在记忆的时间点时，不要只是简单的"什么都没发生"。可以给出一些模糊的、呓语般的文本，暗示主角精神的混沌状态。例如：
    *   `14:30` -> "午后的阳光……太刺眼了。什么都看不清。"
    *   `22:00` -> "安静。世界最好永远这么安静。"

- **游戏时长与难度定位**：
  - 游戏时长的定位大概在2~3小时（读完一本文本量相当的小说的时间）。
  - 谜题实际上不会太难，更多是为了叙事服务。

- **突破性玩法**：
  - **Meta时要求输入设备时间**：在最终meta环节，要求玩家输入现实中的"现在时间"，若玩家输入错误，系统报出当前时间："不对，现在是 XX:XX，对吧？"
  - **可能要求输入的不是时间**：可能要求输入的不是时间，而是日期或者别的密码（也就是说，这个是可以调整的）。比如回答一个不存在的时间，还算是挺有意思。

- **文本质量要求（极简机制的核心支撑）**：
  - 既然机制是极简的，那么每一次成功输入后，反馈的**文本质量**就必须是顶级的。需要确保：
    1.  **高密度信息**：每一个解锁的片段，都不能是废话。它要么提供关键线索，要么渲染极致的氛围，要么展示人物性格的某个侧面。
    2.  **"Aha!"时刻的精心设计**：当玩家通过`10:37` + `辅助时间`解锁A案和B案的区别时，那一刻的文本冲击力必须做到最大。比如，完全相同的场景开头，但在某个关键动词上发生变化（"拉住"变成"推开"），或者内心独白从"不要离开我"变成"你怎么敢离开我"。
    3.  **对"错误"输入的反馈**：见上文"记忆空白与模糊区间"部分。

### 五、叙事诡计与主题提示（写作时的注意点）

- **代词模糊**：尽量用"他/她/那个人/那件事"，延后真名的出现。
- **时间不标日期**：只给出"几点几分"，让不同年份的同一时间在玩家认知中重叠。
- **同名混淆**：A 与 B 同名/同音，E 想成为新的"她"，共同构成"谁配得上这个名字"的暗线。
- **多周目记忆残留**：后期"她"可以偶尔提及玩家此前输入的时间，但不要提前表现出强烈的系统感，留力给最终 meta。
- **现实时间突破**：严格只在最后一次使用系统时间纠正玩家输入，保持冲击力。

- **主题终点的设计理念**：
  *   **不寻求意义**：故事的最终情感落点类似于读完《人间失格》后的体验——那种无法言说、挥之不去的压抑、虚无和对人性深渊的凝视。它并不寻求什么意义，也不提供任何答案或解脱。
  *   **不进行说教**："你也有责任"更多是meta元素的植入，可能一瞬间会给玩家的游戏体验带来负面影响，但这不是说教，而是通过meta元素引发玩家的自我反思。
  *   **不提供审判**：游戏不会给予玩家审判的权力或情感出口。AB没有安息，主角和E的未来是虚无，受害者A和B的悲剧通过主角失去A/B后对C的臆想的强化来体现，而不是通过为她们设计找补的剧情。
  *   **留白与开放循环**：游戏的目标是讲完整个故事，然后留白主角和E的未来以及用户和C的meta交互。最终结局是一个开放的循环，将问题永远地留给玩家。

### 六、语言风格

文本风格参考村上春树和东野圭吾。
