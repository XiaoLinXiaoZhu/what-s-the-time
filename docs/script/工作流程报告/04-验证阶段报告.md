### 3. 转换规则准确性验证

#### 验证方法
- 使用实际示例验证转换规则
- 检查转换后的代码是否符合程序规范

#### 验证示例

**中间格式**：
```markdown
[内心] 早晨的阳光透过梧桐叶片的缝隙，像某种精致的花边装饰。
[停顿:中] 这是新学校的第一周。一切都应该是完美的。

[说] 她指着街角那家新开的包子店，眼睛亮晶晶的。"我们去试试那个吧？听说皮薄馅大，很香的。"

[说] 我看着她，心里涌起一种奇怪的情绪。不是爱——那种东西太虚假了。
[停顿:中] 是一种……占有欲？不对。是保护欲？也不对。
[停顿:中] "你的胃不好，早上吃这么油腻的东西会不舒服的。"

[B] 她眨了眨眼，像是被什么小东西绊了一下。"可是……偶尔试试也没关系吧？"
```

**转换后的程序格式**：
```typescript
{
  type: 'narration',
  text: '早晨的阳光透过梧桐叶片的缝隙，像某种精致的花边装饰。{br}{delay:0.8}这是新学校的第一周。一切都应该是完美的。'
},
{
  type: 'dialogue',
  text: '她指着街角那家新开的包子店，眼睛亮晶晶的。"我们去试试那个吧？听说皮薄馅大，很香的。"'
},
{
  type: 'dialogue',
  text: '我看着她，心里涌起一种奇怪的情绪。不是爱——那种东西太虚假了。{br}{delay:0.8}是一种……占有欲？不对。是保护欲？也不对。{br}{delay:0.8}"你的胃不好，早上吃这么油腻的东西会不舒服的。"'
},
{
  type: 'dialogue',
  character: 'B',
  text: '她眨了眨眼，像是被什么小东西绊了一下。"可是……偶尔试试也没关系吧？"'
}
```

**验证结果**：
- ✅ 文本类型正确转换（`[内心]` → `narration`，`[说]` → `dialogue`，`[B]` → `dialogue` with character）
- ✅ 停顿标记正确转换（`[停顿:中]` → `{br}{delay:0.8}`）
- ✅ 换行正确处理（自然换行 → `{br}`）
- ✅ 符合程序规范（符合 `ScriptSegment` 接口）

**结论**：✅ 转换规则准确，转换后的代码符合程序规范
