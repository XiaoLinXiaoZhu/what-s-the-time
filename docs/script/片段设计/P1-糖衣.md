# 阶段 1: 糖衣 (The Honeymoon)

> **片段数量**: 日常:6-7, 闪回:1  
> **阶段主题**: 建立"虚构恋爱史"的开端；情感绑定；围绕"生日"准备展开  
> **优化版本**: v3.1 (2025-01-27)
> **核心事件**: 她的生日（作为"闹钟"，让玩家知道自己在玩什么）；生日时间的谜题（主角知道但不告诉玩家）；A的生日会（产生怪异感："我的女朋友怎么怪怪的"）

---

## 整体结构设计

### 核心设计理念
- **围绕"生日"展开**：所有片段都服务于"准备生日"这个目标，让玩家知道自己在玩什么
- **生日时间的谜题**：她提到"到时候别忘了来哦"，但主角知道时间不告诉玩家；在准备礼物时看到便签（实际上是A的生日派对时间），引导玩家进入A的生日派对
- **量子叠加态的人际关系**：【主角、A、H】三人小团体 和 【主角、B、D】三人小团体的量子叠加态；他们之间本来就互相认识；A、H关系不错（可能小学就认识了），主角对H有轻微的敌意，但他隐藏得很好，以维系他们三人的关系
- **递进关系**：认识她（B） → 日常互动 → 准备生日 → 看到便签（A的生日时间） → A的生日会（怪异感："我的女朋友怎么怪怪的"）→ 转入P2
- **线索控制**：每个片段只给1个线索，或者不给线索，避免玩家混乱
- **甜蜜为主**：P1主要是甜蜜的，不适的内容只有约5%，在A的生日会上产生怪异感

### 片段结构
1. **P1-DTL-T0805-Breakfast**：认识她（B），她提到下周生日，说"到时候别忘了来哦"（引入核心事件，但时间不明确）
2. **P1-DTL-T0900-MorningWalk**：日常互动，强化情感锚点
3. **P1-DTL-T1230-Lunch**：日常互动，她的朋友（D）提到生日
4. **P1-DTL-T1530-AfternoonTea**：日常互动，她（B）提到生日期待
5. **P1-DTL-T1620-Library**：日常学习，和那个男生（H）的对话；展现【主角、A、H】三人小团体的关系（A、H关系不错，主角对H有轻微敌意但隐藏得很好）
6. **P1-DTL-T1900-Movie**：日常约会，她（B）提到生日计划
7. **P1-DTL-T2145-GiftPlan**：晚上准备生日礼物的独白；看到便签（实际上是A的生日派对时间），主角内心知道但不说（病态控制的展现）
8. **P1-DTL-T2200-BirthdayParty**：A的生日会（玩家看到的是A，而不是B，产生怪异感："我的女朋友怎么怪怪的"；小爆发，5%的不适感，转入P2）

- **ID**: `P1-DTL-T0805-Breakfast`
- **作用**: 建立"虚构恋爱史"的开端；情感绑定；引入核心事件"生日"。
- **进入时间**: `08:05`
- **进入条件**: `FLAG_Game_Start`
- **诡计执行**: 【X合一】。这是**她的真实日常**，但玩家会将其视为"恋爱史"的起点。
- **给予的线索**: [`12:30` (她说"中午在老地方见")] - **只给1个线索**

#### 叙事结构设计（起承转合）

**起（场景建立 - 日常对话）**：
- **情境**：早晨08:05，上学路上，初秋的早晨
- **引入**：主角和她并肩走着，她正在分享昨晚的趣事（电视剧/书/作业等）
- **话题1**：她分享昨晚看的电视剧/书，展现她的兴趣和性格
- **话题2**：她提到今天的课程安排，展现她的日常
- **玩家选择**：主角如何回应她的分享（两个选项，都导向主角的"贴心"回应）
- **目标**：快速建立日常氛围，展现她的形象（纯真、活泼、善良），展现主角的"倾听"和"关注"

**承（事件发展 - 多个桥段）**：
- **桥段1：早餐话题**：
  - 路过新开的包子店，她想吃包子，表现出期待
  - 主角已经提前准备好了早餐（豆浆+包子/其他）
  - 她的反应：先是惊喜，然后有一丝困惑（"你什么时候准备的？"）
  - 主角的回应：用"贴心"包装（"我担心你早上会饿"）
  - **玩家选择**：主角如何解释（两个选项，都导向"贴心"）
- **桥段2：生日话题（核心事件引入）**：
  - 她突然提到："对了，下周我生日，你有空吗？到时候别忘了来哦。"
  - 主角的内心独白：解读她的邀请，认为"她需要我"；主角知道生日的时间，但不告诉玩家
  - 主角的回应：展现"承诺"和"期待"（"当然，我会为你准备一个特别的礼物"）
  - 她的反应：笑容甜美，但有一丝疲惫（5%的不适感）
  - **关键**：她没有说具体时间，只说"到时候别忘了来哦"，主角知道但不说，埋下时间谜题
- **桥段3：中午约定**：
  - 她提到"中午在老地方见"，主角点头
  - 主角的内心：对"老地方"的占有感，认为这是"我们的地方"
- **目标**：通过多个桥段展现主角的"贴心"和"关注"，同时引入核心事件"生日"，让玩家知道自己在玩什么

**转（转折点 - 深度挖掘）**：
- **转折1**：她突然问主角："你为什么对我这么好？"
- **主角的内心独白**：揭示真实动机（"因为她需要我"、"因为她是我的人"）
- **主角的回应**：表面上是"因为我喜欢你"，但内心是控制欲
- **转折2**：她听到主角的回答，笑容变得微妙，有一丝疲惫（5%的不适感）
- **主角的观察**：注意到她的疲惫，但自我说服"她只是累了"
- **转折3**：她提到生日，主角承诺准备特别礼物
- **主角的内心**：对"特别礼物"的期待，认为"她应该喜欢我准备的礼物"

**合（结果与悬念 - 单线索释放）**：

- **伏笔**：
  - 她提到下周生日，说"到时候别忘了来哦"，但时间不明确，这是整个P1阶段的"闹钟"和谜题
