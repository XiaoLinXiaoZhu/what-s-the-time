### 简要说明

你现在已经有了**按真实时间线**的切片设计（`时间线.md + 切片设计.md`），你这次要的是：  
**按“叙事诡计后的呈现顺序”重排整个故事流程**，让玩家在体验上先看到“一个 X 的故事”，再逐步拆分成 A/B 两起案件。

下面是一个**完整的游戏流程故事设计**：按玩家体验顺序，而不是按真实时间线。

---

## 一、设计目标再抽象（叙事层重排的方向）

- **X 合一 → X 拆分**  
  - 玩家前期只认知到一个“她 = X”，她：  
    - 有一段长时间的亲昵日常；  
    - 有觉醒与质疑；  
    - 最终约谈、死亡。  
  - 这些内容实际上是：  
    - **A & B 的日常 → B 的觉醒 → A 的约谈 → A 的死亡 + B 的死亡（隐去）**  
  - 即：**把 B 线前半段嫁接到 A 线前半段，拼成一条“虚构的单一恋爱史”。**

- **主角自我洗脑：把 B 替换为 A**  
  - 在主角的叙事里，第二起事件中“B 的消失”被讲述为“X 一年前的死亡后续阴影”，  
  - 大量与 B 相关的细节被重新贴到“那位 X（A）”身上，从而“说服自己 → 只有一件事、只有一个人”。

- **C（理想她）+ E 的穿插**  
  - 在关键节点插入与“理想她 C”的白房间对话、与 E 的暧昧对话，  
  - 让玩家以为“理想她 + 现实她 + 过去她”都是同一个模糊人，  
  - 同时用 E 作为“旁观者/共犯”，慢慢透露：其实有**两次**、**两个人**。

- **时间诡计：同一时间，不同事件**  
  - 如 `10:07 / 11:00`：  
    - 一次是 A 案楼顶 + A 案问话；  
    - 一次是 B 案对峙 + B 案失踪问话。  
  - 游戏用“两问一锁”机制，先只让玩家看到“混合版 X 事件”，再用组合键拆出 A/B 版本。

- **人物混淆：A/B/C，H/D 故意糊成一团**  
  - 前中期：  
    - 所有 X（A/B/C 位）统称为“她 / X”；  
    - H/D 都是“那个人”“那位同学”“总在旁边说话的人”；  
  - 后期：通过矛盾细节+多视角碎片，让玩家发现“这里其实是不同的人”。

- **非线性探索：支线与多线索片段**  
  - 设计一批“**一段里带两、三个矛盾线索**”的切片，  
  - 玩家不会线性顺序刷完，而是在矛盾中来回跳转，自己组装真实时间线。

## 二、宏观流程：按玩家体验划分的 4 个阶段

### 阶段 1：X 的恋爱故事（合并后的单一叙事）

**玩家认知：**  
“我操控了一个曾经有女友 X 的可怜人，她死了，现在我在回忆这段唯一的恋爱。”

#### 1.1 开场 & 时间机制教学（Loop 0）

- 开场：  
  - 模糊之声问：“时间是……？”  
  - 主角接话：“不，是他在问我。”（轻微别扭墙）  
  - 玩家被要求输入任意 `HH:MM`，初次输入被引导到 `08:05`。

- 教学时间点：  
  - `08:05`：**混合 A/B 的“X 的早晨”**  
    - 表面上：和“女友 X”的甜蜜日常（早餐、上学路）。  
    - 实际上：  
      - 部分内容来自 A 的旧校早餐（护手霜、老教学楼）；  
      - 部分来自 B 新校上课路（贩卖机、汽水）。  
    - 主角叙述把两组环境混成一个模糊场景：  
      - 楼道既像旧校水泥，又有新校油漆味；  
      - 桌子既有护手霜，也有汽水罐。  
    - 玩家只得到印象：“他们在一起走了很久，从高中过渡到大学一样。”

- 其他早期时间点（线性上前后，但呈现时不强调年份）：  
  - `07:50`、`08:20` 等  
    - A/B 的课堂、社团、跟朋友打闹、被调侃像情侣，这些全部统称“X 与他身边的那个朋友（H/D 混像）”。  
    - 对“那个朋友”的描写故意模糊脸与称呼，有时更像 H、有时更像 D，统一叫“那个人”。

**结果：**  
玩家建立了一个**连续单一的“X 恋爱史”**：  
- 他们认识很久、一起上学、一起吃饭；  
- 身边有个总爱吐槽的“那个人”（H/D 混合体）；  
- 主角看上去只是有点依赖/控制欲，不像真凶。

---

### 阶段 2：X 的不安与觉醒（主要抽 B 的素材）

**玩家认知：**  
“X 渐渐觉得这段关系不对，想和我‘好好谈谈’，后来……她死了。”

#### 2.1 X 的觉醒（以 B 的觉醒为主，讲成 X 的中后期）

- 时间例子：`12:10`、`21:10` 等  
  - `12:10`：  
    - 教室或社团间，X 认真说“我们该谈谈”，  
    - 内容实际来自：  
      - A 的首次认真沟通 + B 的“由内而外的直觉质疑”，  
      - 但串成一条“持续多次的谈谈”。  
  - `21:10`：  
    - 短信草稿：“我总觉得你看我的时候，在看……算了。”  
    - 实为 B 写给主角的未发消息，但主角叙述时说“那是她当年写的”。

- 支线片段：  
  - “专属物品”错乱：  
    - 一段写“她喜欢柠檬味护手霜”；  
    - 另一段写“她老爱喝自动贩卖机的汽水”；  
    - 主角却把两者描述成同一个人的稳定习惯，玩家前期只会觉得“她很复杂”。

#### 2.2 朋友的提醒（H/D 混合）

- 几个时间点重复出现类似场景：楼道/天台楼梯口，“那个人”在对 X 说：  
  - “你们这样不健康。”  
  - “他总是在替你作决定。”  

- 叙事诡计：  
  - 第一次，描述略像 H（暴躁、口气冲）；  
  - 第二次，同一时间点被第二次访问时，内容略像 D（更冷静、关心 B）；  
  - 主角都称呼对方为“那个人”，玩家以为是同一人，只是情绪不稳定。

**结果：**  
- X 的觉醒 = B 的觉醒 + A 的理性思考，被合并成一个“长期忍耐后决定谈一谈”的发展线；  
- H 与 D 的劝告被混为一个“朋友视角”，加强“旁观者其实早就看出了问题”的感受，但不分是谁。

---

### 阶段 3：X 的约谈与死亡（“A+B 拼成一场死亡”）

**玩家认知：**  
“有一天（十点零七），X 约我/或我去见她，我们在天台/教室激烈争吵，她掉下去了。我被问话，那是一场唯一的悲剧。”

这里是**叙事重排的核心**：

#### 3.1 10:07：合成的“X 坠落事件”（初始版本）

- 输入 `10:07`（第一周目 / 前期）：  
  - 场景是一个模糊的高处空间：  
    - 护栏细节来自 A 案楼顶；  
    - 部分视觉元素来自 B 案的空教室或新教学楼天台；  
  - 对话：  
    - 前半段引用 B 对“替身感”的控诉：“你看着我的时候，在看谁？”  
    - 后半段引用 A 在“理性谈判”时的台词：“我们不能再这样下去了。”  
  - 最后：  
    - 只表现“她后退一步，身体失衡”，“空白”，“坠落感”，  
    - 不明确到底是被推、自己退、还是意外。  

- 玩家看到的，是**一场综合了 A/B 的“X 之死”**，时间锚点为唯一的 `10:07`。

#### 3.2 11:00：合成的“X 案问话”（初始版本）

- 输入 `11:00`（自然顺延）：  
  - 问话场景也是**旧案+新案问话的合体**：  
    - 有警官 G；  
    - 有老师 F 在旁说“他有创伤后遗症”；  
    - 问题台词同时夹带：  
      - A 案中“那天早上你们去哪儿”；  
      - B 案里“她最近有没有说要离开”。  
  - 叙述人只说：“那天，他们都在问关于 X 的问题。”  

- 玩家感觉：  
  - “这就是当年那场惨剧的唯一问话流程”，完全不会想到一年后还有一模一样的一次。

#### 3.3 C（理想她）对白的穿插

- 在 10:07 / 11:00 的“混合版”之后，会偶尔出现一个**无时间坐标的白房间片段**：  
  - 主角与一个模糊轮廓的“她”对话，这个她：  
    - 既像 A，又像 B；  
    - 台词又像 E 在安慰他。  
  - 她被主角在叙述中称为“C”或“真正的她”，  
  - 内容是：“你不是故意的”“他们都误会你了”——帮主角完成自我洗脑。

**关键点：**  
- 到这一步，玩家心里只有一个故事：  
  - **一个 X，从甜蜜 → 不安 → 约谈 → 坠楼 → 案发问话**。  
  - 时间线在感受上是**单线**。

---

### 阶段 4：裂缝出现——支线与多线索片段让 X 变得“不可能只有一个人”

从这里开始进入“探索 / 非线性”的主场，通过**支线 + 多线索片段**打破单一 X 叙事。

#### 4.1 多线索片段示例（每段里塞两个互相打脸的信息）

- 片段类型 A：**记忆错位的礼物**  
  - 某个时间（如 `夕阳放学时间`）玩家看到：  
    - 主角回忆给“她”送生日礼物：  
      - 画面中既出现 A 用过的护手霜礼物包装；  
      - 又出现 B 喜欢的汽水瓶吊饰；  
    - 叙述却说“那是她唯一一次收礼物”。  
  - 玩家开始觉得：**一个人不太可能同时有这两套稳定喜好**。

- 片段类型 B：**H/D 混合视角**  
  - 某个输入（例如 `某天下午 16:20`）触发一段“旁观者日记”：  
    - 日记里写：  
      - “他看她的眼神不对劲，我得提醒她。”  
      - 有的内容明显是 H 在说 A（文风粗糙，地点是旧校天台）；  
      - 有的内容更像 D 在说 B（文风细腻，地点是新校社团室）。  
    - 署名被处理为 `{blur}H{/blur}` / `{blur}D{/blur}`，看不清。  
  - 玩家开始怀疑：**是不是有两个人在不同时间提醒过两段不同关系？**

#### 4.2 两问一锁机制上线：同一时间点的“双解”

- 在玩家多次访问 `10:07`、`11:00` 后，系统开始提示：  
  - “记忆不稳定，可能有另一层叠加，请输入一个‘那天开始不对劲的时间’。”  

- 例子：  
  - `(10:07, 08:05)` → 显示更完整的 A 案那天（有 H、旧校天台、栏杆手脚）；  
  - `(10:07, 18:40)` → 显示 B 案当天对峙+ E 成为共犯（新校环境、贩卖机、黑色塑料袋）；  
  - `(11:00, 10:07 A)` vs `(11:00, 10:07 B)` → 两次问话镜像版本。  

**玩家第一次清楚感知：**  
- “原来**同一个时间点，实际上是两天、两件事**。”  
- 之前看到的“混合版 10:07/11:00”，只是这两件事叠在一起的**主角自我改写结果**。

#### 4.3 X 的拆分：从“一个人”变成“两起案件”

- 当玩家解锁足够多的“双版本片段”后，系统可以给出一次“X 专场”：  
  - 将所有涉及 X 的片段按**环境 & 感官锚点**分类：  
    - 柠檬护手霜 + 旧水泥护栏 → A；  
    - 汽水声 + 新教学楼油漆味 → B；  
  - 玩家亲手把属于 A 的片段拖到一边，把 B 的拖到另一边（即便在 UI 上只是文字列表、逻辑上已经分组）。  

- 叙事上：  
  - 主角开始口吃、改口：“她……她们……她。”  
  - C（理想她）的对白变得更激烈：“你明明知道有两次，却还要把她们拼在一起，这样就能减轻一点罪吗？”

## 三、真相周目：两条真实时间线被强制并排播放

**玩家认知转为：**  
“原来是两件事、两个 X、一堆共犯和替罪羊；而主角一直在用自己的叙事把她们揉成一个‘故事’压在我脑子里。”

### 3.1 A 线真相：第一次杀人 + 栽赃 H

- 用 `(10:07, A 标记)`、`(11:00, A 标记)` 等组合，播放**完整 A 案真相版**：  
  - A 的理性觉醒、纸条约谈、天台手脚、H 的善意被利用；  
  - 问话中主角如何一步步把 H 塞进 C 位。  

### 3.2 B 线真相：重复犯罪 + E 成为共犯

- 用 `(10:07, B 标记)`、`(18:40, B 当晚)`、`(11:00, B 失踪问话)`，播放**完整 B 案真相版**：  
  - B 的“我不是她”对峙；  
  - 主角更冷静的暴力；  
  - E 的“最后机会”，以及“叫她 C”的堕落宣言。  

### 3.3 C 位与“怪物位置”的揭示

- 探视室、白房间、Meta 对话中：  
  - 清晰点名：  
    - H 曾被社会故事当成 C；  
    - A 是主角心中的最初 C；  
    - B 被强行塞进 C 模具；  
    - E 选择成为“最后的 C”；  
  - 玩家被问：“你看故事时，也在找一个 C 来承受一切，对吗？”

## 四、Meta 终局：玩家被拉入叙事责任链

- 多周目后，主角开始直接对玩家抱怨 `10:07` 被反复打开（S33）：  
  - “又是这个时间。你到底想看什么？你是想看她，还是想看她们？”  

- 最终 Meta 场景（S34）：  
  - “那就说说，现在几点。”  
  - 玩家输入现实时间，被系统纠正。  
  - 文本把**现实时间 / 游戏内当前周目时间 / 手表停在的 10:07**并列，  
  - 主题落点：  
    - 主角通过叙事把两次暴力合成一个“悲剧故事”；  
    - 玩家通过游玩不断回放这两次暴力；  
    - 故事需要很多人一起才能成为“故事”。

---

## 五、非线性探索设计要点（补充）

- **支线片段**  
  - 每条支线不只是“补一块真相”，而是**同时提供两条彼此冲突的信息**（A/B or H/D 的混用），  
  - 只有在真相周目玩家开始拆分时，这些支线才会“对号入座”。

- **多线索片段**  
  - 一段里的感官、环境、称呼、人际关系故意“跨案混用”：  
    - 旧校环境 + B 的语言风格；  
    - 新校环境 + A 的理性台词；  
  - 在前期都被解释为“记忆模糊”；  
  - 在后期玩家意识到：这不是模糊，而是主角主动的**混剪**。

---

**总结一句话：**  
真实世界里有**两起案件、两位 X、一堆旁观者与共犯**；  
游戏中，玩家先被迫接受一个**被主角重新剪辑过的“单一 X 的爱情悲剧”流程**，  
再通过时间输入、多版本片段、支线矛盾，亲手把这条“合成叙事”拆开，看到它原本是两条被暴力拼起来的生命线。
