*初秋的早晨，空气里有一种类似清冷的、透明的质感。*

她走在我身边，正在分享昨晚看的电视剧。声音轻快，像某种清脆的、会跳跃的音符。

"你看了吗？那个男主角最后的选择，我觉得太可惜了。明明可以在一起的。"

我听着，偶尔点头。心里却在想别的事情。

*不是爱——那种东西太虚假了。*

*是一种……占有欲？不对。是保护欲？也不对。*

*大概，只是因为她需要我。*

"听说了吗？那个男生又被叫去问话了。"

"哪个？"

"就是那个H啊。总是惹事，脾气差得像疯狗一样。"

*H。*

*那个名字像某种尖锐的东西，刺破了我的意识。*

她突然停下脚步，好奇地问："那个男生是谁？"

*她不应该知道。*

*她不应该好奇。*

*她不应该……*

"就是那个总是惹事的家伙。"一个同学回答，"听说昨天又在图书馆跟人起冲突了。"

*她不应该去那里。*

*那里是……*
