*早晨的阳光透过梧桐叶片的缝隙，像某种精致的花边装饰。*

*这是新学校的第一周。一切都应该是完美的。*

她走在我身边，正在分享昨晚看的电视剧。声音轻快，像某种清脆的、会跳跃的音符。

"你看了吗？那个男主角最后的选择，我觉得太可惜了。明明可以在一起的。"

我听着，偶尔点头。心里却在想别的事情。

不是爱——那种东西太虚假了。

是一种……占有欲？不对。是保护欲？也不对。

*大概，只是因为她需要我。*

她突然停下脚步，指着街角那家新开的包子店，眼睛亮晶晶的。

"我们去试试那个吧？听说皮薄馅大，很香的。"

*那种期待的表情。*

*像某种小动物，在等待投喂。*

我看着她，心里涌起一种奇怪的情绪。不是爱——那种东西太虚假了。

是一种……占有欲？不对。是保护欲？也不对。

"你的胃不好，早上吃这么油腻的东西会不舒服的。"

她眨了眨眼，像是被什么小东西绊了一下。

"可是……偶尔试试也没关系吧？"

"偶尔？"我笑了笑，去旁边的便利店买了三明治。

"你看，这个蛋白含量高，营养更均衡。"

她接过三明治，咬了一口，然后露出一个标准化的笑容。

"嗯，很好吃。"

*那种笑容。*

*我见过。在很久很久以前，另一个女孩也露出过这样的笑容。*

*那种被训练出来的、迎合式的笑容。*

*但她的笑容里，似乎还有一丝……疲惫？*

*大概是昨晚没睡好吧。*

她突然想起什么，转过头看着我。

"对了，下周我生日，你有空吗？到时候别忘了来哦。"

*生日。*

*我当然知道是哪一天。我当然会去。*

*但我不会告诉她具体时间。*

*让她自己记住，让她自己期待。*

*这样，她才会更依赖我。*

"当然，我会为你准备一个特别的礼物。"

她的笑容更甜了，但那一丝疲惫似乎更深了。

*她为什么疲惫？昨晚为什么没睡好？*

*大概，只是学习太累了吧。*

"中午的时候，我们还是去那个老地方吧。"她走在前面，声音里没有任何起伏。

"还是老地方。"我重复着，仿佛在确认什么。

*老地方。*

*那是我们的地方。*

*只有我们两个人知道的地方。*

*12:30。*

*中午12:30，在老地方。*

*我会记住的。*

*但她不需要知道，我已经记住了。*

*我看着她的背影，想起了一个问题。*

***她今天露出的疲惫笑容，是从哪里学来的？***
