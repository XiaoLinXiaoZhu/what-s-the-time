## 1. 文风检查清单

### 1.1 支柱一：不可靠的第一人称

**核心定义**：主角的叙述是经过自我美化和扭曲的"他的真实"，他说的未必是谎言，但永远是经过自我美化和扭曲的"他的真实"。他通过"辩解"来维护自己的"受害者形象"，将责任转嫁给他人。

**检查项**：
- [ ] 使用自问自答（"我为何要那么做呢？大概是因为…"）
- [ ] 淡化恶意，强调懦弱和恐惧（不说"我出于恶意"，而说"我因为害怕失去，才做了那件无可奈何的事"）
- [ ] 对他人行为的病态解读（将B的觉醒解读为"她不再纯粹了"，将D的关心解读为"多管闲事的窥探"）
- [ ] 频繁使用"大概"、"或许"、"似乎"等不确定词汇
- [ ] 通过对比"旁人看来"和"我的真实"，突出主角的不可靠性
- [ ] 时常用"他"指代"当年的那个我"，以此切割责任

**示例**：
- ✅ 好："我为何要那么做呢？大概是因为...我只是害怕失去她。"
- ✅ 好："旁人看来或许很奇怪吧。但我只是...我只是想保护她。"
- ❌ 差："我很愤怒，所以我杀了她。"
- ❌ 差："我清楚地知道我在做什么。"

### 1.2 支柱二：疏离感

**核心定义**：主角无法真正融入世界，他的一切行为都是在"扮演"一个正常人。他对周围的一切都有一种抽离的、冷漠的观察感。

**检查项**：
- [ ] 感官描写的"失焦"（不说"阳光明媚"，而说"某种刺眼的光线从窗外渗进来，让我感到一阵眩晕"）
- [ ] 将情感"物体化"（不直接说"我很难过"，而说"一种类似悲伤的、沉重的东西，盘踞在我的胸口"）
- [ ] 引用他人的"正常"来反衬自己的格格不入（"'普通人'在这种时候，似乎应该会哭吧。但我一滴眼泪也流不出来。"）
- [ ] 使用"某种"、"类似"、"仿佛"等模糊化词汇
- [ ] 通过对比"他们"和"我"，突出主角的异质性

**示例**：
- ✅ 好："某种刺眼的光线从窗外渗进来，让我感到一阵眩晕。"
- ✅ 好："一种类似悲伤的、沉重的东西，盘踞在我的胸口。"
- ❌ 差："阳光明媚，我心情很好。"
- ❌ 差："我很难过，想哭。"

### 1.3 支柱三：耻感与自我厌恶

**核心定义**：尽管主角不断辩解，但在其内心深处，是对自己存在的强烈"耻感"和厌恶。这种情绪不是以外显的忏悔，而是以自嘲、自我贬低的形式出现。

**检查项**：
- [ ] 突兀的自我嘲讽（在一段看似正常的叙述后，突然插入一句自我否定的内心独白）
- [ ] 对赞美的恐惧（当被人夸奖时，主角的内心反应不是喜悦，而是恐慌）
- [ ] 在看似正常的叙述后，突然插入括号内的自我嘲讽
- [ ] 通过自我贬低，揭示主角内心的真实状态

**示例**：
- ✅ 好：在描写完自己如何体贴B之后，内心独白："（看呐，一个多么完美的、体贴的男友。）"
- ✅ 好："他们称赞的那个'我'，根本不存在。"
- ❌ 差："我为自己感到骄傲。"
- ❌ 差："我确实是个好人。"

### 1.4 支柱四：语言的"表演性"

**核心定义**：文风是颓废的，但绝不粗糙。它有一种经过精心打磨的、病态的美感。用词讲究，句式从容，即使在描述最不堪的情感时，也保持着一种奇异的优雅。

**检查项**：
- [ ] 避免口语化、网络化的词语（除非是引用他人对话）
- [ ] 多用比喻，尤其是自然或物理比喻（"我的谎言像藤蔓一样，缠绕住她，也缠绕住我自己"）
- [ ] 保持书面语的精致
- [ ] 即使在描述最不堪的内容时，也要保持语言的优雅

**示例**：
- ✅ 好："我的谎言像藤蔓一样，缠绕住她，也缠绕住我自己。"
- ✅ 好："这份爱，像一块冰，被我死死攥在手心，既不愿它融化，又被冻得生疼。"
- ❌ 差："我骗了她，这让我很内疚。"
- ❌ 差："我太爱她了，不想失去她。"

### 1.5 技法一：反复/纠结/对比表现

**核心定义**：分行显示才有张力。当主角在两个念头之间徘徊、反复追问、对比矛盾时，必须让句子分行显示，形成上下对照的视觉节奏。

**检查项**：
- [ ] 自我追问分行显示（`时间不对。哪里不对？`）
- [ ] 对比怀疑分行显示（`现在是……这个时间吗？还是说，那只是某一次……坠落的时间？`）
- [ ] 递进强调分行显示（`只有风声，只有红色，只有……`）
- [ ] 反复尝试分行显示（`我试图拼凑，试图抓住，但只有虚无。`）
- [ ] 使用省略号营造不确定感

**示例**：
- ✅ 好：
```
时间不对。

哪里不对？
```

- ✅ 好：
```
占有欲？不对。

是保护欲？也不对。
```

- ❌ 差："时间不对，哪里不对？占有欲不对，是保护欲也不对。"（没有分行，缺乏张力）

### 1.6 技法二：比喻与意象策略

**核心定义**：画面感优先于概念。将比喻前置，先给出具象画面，然后展开动作。

**检查项**：
- [ ] 比喻前置（先给出具象画面，如"像颜料一样"，然后展开动作）
- [ ] 使用自然现象和物理现象的比喻
- [ ] 通过动作词（"晕开"、"蔓延"、"吞噬"）强化画面感
- [ ] 避免抽象概念先行

**示例**：
- ✅ 好：
```
模糊的红色，像颜料一样。

在视野里晕开，蔓延，吞噬一切。
```

- ❌ 差：
```
模糊的红色。

像颜料一样，在视野里晕开...
```
（"模糊的红色"过于抽象，玩家没有画面感）

### 1.7 技法三：精简与节奏的平衡

**核心定义**：去除冗余，保留必要的呼吸。叙事应该流畅有张力，应该按照叙事的节奏一张一弛。

**检查项**：
- [ ] 删除重复的追问（三个变两个）
- [ ] 删除冗余的形容词（"那个时间，那个瞬间" → "那个瞬间"）
- [ ] 保留反复强调的核心词（"时间不对"出现两次，第二次用强调）
- [ ] 保留递进的"只有"结构（风声/红色/...）
- [ ] 在关键情绪节点使用自然停顿（通过换行实现）

**示例**：
- ✅ 好：保留核心信息，去除冗余
- ❌ 差："那个时间，那个瞬间，那个时刻，我永远忘不了。"（冗余）

## 2. 人物特征清单

### 2.1 主角（"我"）

**核心心理机制**：病态的自我合理化 + 极端控制欲

**必须符合的特征**（硬约束）：
1. **不可靠叙述**
   - 言行模式：频繁辩解、自问自答、将责任转嫁
   - 示例：用"他"指代"当年的我"；用"她"回避具体名字
   - 语言特点：时常用"他"指代"当年的那个我"，以此切割责任；对A/B几乎都只用"她""那个人"，拒绝正面说出全名

2. **控制欲的展现**
   - 言行模式：替她做决定、修正她的品味、排斥她的社交
   - 示例："你的胃不好，吃这个更健康"（实则是控制）
   - 示例：B分享她喜欢的乐队，主角说："我明白了。但我觉得，你听这首古典乐时，会更美。"（不是在推荐，而是在"修正"B的品味）

3. **完美面具（创伤后遗症表演）**
   - 言行模式：表现出"脆弱""痛苦""无法控制自己"，让所有人相信他是无辜的
   - 示例：在审讯中表现出创伤后应激（PTSD）、极度自责、记忆混乱
   - 关键：这个"创伤后遗症"的形象是他精心维护的完美面具

4. **推责与叙事操控**
   - 言行模式：不仅对他人推责（如栽赃给H），也会对玩家推责
   - 示例：在回忆中重构虚假选择，让玩家误以为自己做出了"杀死B"的决定
   - 示例：通过嘲讽玩家，将整件事情的责任推给玩家："是你选的。""是你让她死的。""不是我，是你。"

**建议体现的特征**（软约束）：
- 对时间的执着（10:37）
- 对完美的追求
- 语言的表演性（精致的颓废，书面语，比喻密集）
- 对赞美的恐惧
- 突兀的自我嘲讽

### 2.2 A（旧案受害者，名字：X）

**核心特征**：独立、有主见、理性的反抗者

**必须符合的特征**（硬约束）：
1. **独立与有主见**
   - 言行模式：有明确的人生规划和目标，不依赖他人做决定
   - 示例：习惯独立思考，对不公或控制行为敏感
   - 示例：重视真实与诚实，厌恶被当作"所有物"

2. **理性的反抗**
   - 言行模式：收集证据、带着论据和决心，像一场谈判
   - 示例：可能收集了主角控制欲的证据（比如聊天记录、日记）
   - 示例：她的"好好谈谈"是带着论据和决心的，像一场谈判
   - 与B形成对比：A的反抗是"理性的"，B的反抗是"感性的"

3. **觉醒：由外而内**
   - 言行模式：H的提醒是关键催化剂
   - 示例：她从朋友的警告中开始审视自己的关系，是一个相对被动的过程
   - 与B形成对比：A的觉醒是"由外而内"，B的觉醒是"由内而外"

4. **专属物品（用于区分A与B）**
   - 示例：A喜欢某个牌子的护手霜（例如：柠檬香味的廉价洗手液）
   - 示例：A喜欢喝茶
   - 关键：在真相周目，当玩家输入特定时间，会看到清晰的记忆：主角将A的护手霜放在B的桌上，B困惑地说："我不喜欢这个味道。"

**建议体现的特征**（软约束）：
- 优等生、独立，有些人觉得她"太有主见/太强势"
- 重视关系的平等和尊重
- 对控制行为有敏锐的察觉力

### 2.3 B（现案受害者，名字：X）

**核心特征**：随性、阳光、直率，感性的反抗者

**必须符合的特征**（硬约束）：
1. **感性的反抗**
   - 言行模式：凭直觉感到不对劲，直接的质问
   - 示例：她的"好好谈谈"更像是一种情绪的爆发，是直接的质问："你到底是谁？你看着我的时候，在看谁？"
   - 与A形成对比：B的反抗是"感性的"，A的反抗是"理性的"

2. **觉醒：由内而外**
   - 言行模式：更多是依靠自己的感受和观察
   - 示例：D可能会提供一些侧面信息，但B的决定性怀疑来自于她与主角的直接互动
   - 与A形成对比：B的觉醒是"由内而外"，A的觉醒是"由外而内"

3. **专属物品（用于区分A与B）**
   - 示例：B喜欢喝某个牌子的汽水（例如：自动贩卖机里的冰镇可乐）
   - 关键：在真相周目，当玩家输入特定时间，会看到清晰的记忆：主角买了B爱喝的汽水，却下意识地对她说："你以前不是喜欢喝茶吗？"（A喜欢喝茶）

4. **同情的来源**
   - 言行模式：在新学校听说了主角"一年前那件事"（A案），对主角产生同情
   - 示例：起初真心把主角当"需要照顾的可怜人"，有一段短暂的好感期

**建议体现的特征**（软约束）：
- 在现实性格上明显不同（例如更随性/更阳光/更直率）
- 自己有明确的人生目标（考学/离开小镇/追求某种生活）
- 对被他人"束缚"非常敏感

### 2.4 E（一年前的目击者 + 现案共犯）

**核心特征**：扭曲的自我实现，试图成为"最后的C"

**必须符合的特征**（硬约束）：
1. **共犯动机的复杂性**
   - 言行模式：将犯罪行为视为一种"升华"，一种摆脱平庸、与心爱之人建立终极连接的仪式
   - 示例：当她目睹罪行时，她面临一个选择：报警，回归平庸；或者，加入他，让这份禁忌的爱成为自己人生的"主题"
   - 关键：她的选择不是简单的"爱情蒙蔽双眼"，而是将犯罪行为视为一种"升华"

2. **"最后的机会"场景**
   - 言行模式：在B案后，E在主角处理现场前，可以通过文本描述来展现她的内心挣扎
   - 示例："E的手伸向口袋，指尖触碰到手机的冰冷外壳。屏幕上，三个数字已经按了下去：1-1-0。但就在这时，她看到了——不是冷酷，不是残忍，而是一种迷茫和脆弱，像是一个迷路的孩子。她的手停住了。手机滑回口袋。她走向了他。"

3. **"叫她C"的宣言**
   - 言行模式：E主动说"叫她C"，试图占据"替罪羊"的位置
   - 示例："她说，叫她C。""她会帮他处理掉C。"
   - 关键：E自称C，但她的真实名字不是X，这样三个"她"（A、B、E）中只有A和B的名字是X

4. **连接前后两件事的证人**
   - 言行模式：一年前在高中得知了主角做了手脚（但没有看到推搡过程），一年后在新学校目击了B案（清晰目击）
   - 关键：E因为喜欢主角而选择同一所新学校，成为唯一一个同时见证两起案件的人

**建议体现的特征**（软约束）：
- 长期在心理上"站在主角这边"，想成为那个"真正理解他的人"
- 她的爱里有真实的心动，也有逃避
- 在最终meta环节，E与主角内心创造出的理想化"她"（C）融合

### 2.5 H（旧案中的替罪羊）

**核心特征**：完全无辜，被社会偏见和主角栽赃的牺牲品

**必须符合的特征**（硬约束）：
1. **"标签先行"的展现**
   - 言行模式：在玩家第一次听到H的名字时，就伴随着各种负面标签
   - 示例："那个暴躁的家伙"、"总打架的H"
   - 关键：这些信息由看似可靠的角色（如老师F、同学）说出

2. **对A的好感**
   - 言行模式：H本身对A有好感，但从未明确表达
   - 关键：这种好感可能让主角将他视为"威胁"

3. **H的善意提醒**
   - 言行模式：H察觉到A和主角关系的问题，主动告诉A这段关系不健康
   - 示例：H提供了一些证据（具体内容可后续细化）
   - 关键：主角刚好路过，听到了H对A说的话，但假装不在意

4. **被主角利用的陷阱**
   - 言行模式：H因为担心A的安全，急忙赶到天台
   - 关键：H的动机是保护A，但主角利用了这个善意，让H成为"凶手"

**建议体现的特征**（软约束）：
- 有"坏名声"（暴躁、打架、逃课等标签）
- 在一年前的舆论中被迅速判定为"最可疑的人""根本不正常"
- 与A有普通同学/朋友关系

### 2.6 D（B的朋友/同学）

**核心特征**：为B说话，看穿主角的谎言

**必须符合的特征**（硬约束）：
1. **核心驱动力是"为B说话"**
   - 言行模式：大家都说"她可能只是有事回家"，只有D坚定认为"她不会不打招呼就消失"
   - 示例：D在整理B的遗物时，发现了B的日记，里面记录了她对主角的怀疑和不安

2. **"菠萝包"证据**
   - 言行模式：B曾对D说过："如果我哪天给你发一个'菠萝包'的表情，就说明我又在应付那个家伙了，我烦透了。"
   - 示例：B失踪前发给D的最后一条消息里，正好带着那个"菠萝包"
   - 关键：D的质问变得更加具体而尖锐："B失踪前，给我发了'菠萝包'。你知道那是什么意思吗？"

3. **从友好到怀疑的弧光**
   - 前期：调侃主角和B像情侣，帮他们打圆场
   - 中期：逐渐收集矛盾信息，对主角态度从中立 → 怀疑
   - 后期：明确质问主角："那天你到底对她说了什么？"

**建议体现的特征**（软约束）：
- B的好友/室友/社团同伴，对B的日常状态最敏感
- 与主角有普通同学关系，前期对他并无敌意
- 可设定为：要么暗恋B，对主角有潜在敌意；要么将B当作最重要的朋友，对任何伤害她的人零容忍

### 2.7 F（班主任/导师）

**核心特征**：从同情到不安，代表"制度/成年人"的视角

**必须符合的特征**（硬约束）：
1. **前期：把主角所有怪异行为都解释为创伤后遗症**
   - 言行模式：一开始对主角极其同情，帮他挡掉很多麻烦
   - 示例：F老师以"创伤后遗症"为由不断替主角说情

2. **变化轨迹**
   - 前期：把主角所有怪异行为都解释为创伤后遗症
   - 中期：隐隐不安，开始注意到他讲述中的矛盾
   - 后期：面对警方与学校压力，摇摆在"保护学生"和"追求真相"之间

**建议体现的特征**（软约束）：
- 负责主角班级/课题，肩负"教育者"的位置
- 在多个问话场景中出现，代表"制度/成年人"的视角

### 2.8 G（警官）

**核心特征**：不夸张、不戏剧化，语气平静但有压迫感

**必须符合的特征**（硬约束）：
1. **重复的问话**
   - 言行模式：一年前旧案、现在失踪案都参与过的警官，一直记得主角这张脸
   - 示例：在问话室中使用大量模糊代词："那件事""她""那时候"，让旧案与新案在语言层面缝合
   - 关键：对玩家而言，是"时间与法律"的代表人物：你可以躲开记忆，但躲不开他

2. **问话风格**
   - 言行模式：不夸张、不戏剧化，语气平静但有压迫感
   - 示例：G警官记录时，笔尖的犹豫
   - 示例：G警官看主角的眼神，从同情变成了怀疑

**建议体现的特征**（软约束）：
- 在A案和B案中都参与问话
- 最后一次谈话，G对主角说"你可以回去了"，主角看表，表盘仍是10:37

### 2.9 C（概念性"位置"）

**核心定义**：C不再是某个具体人的名字，而是一个"角色位"：
- 那个主角心里"可以为我做任何事的人"
- 那个被社会故事认定为"那件事中的怪物/关键人物"的位置

**占据过/试图占据C位的人**：
- **H（旧案中的替罪羊）**：被社会故事塞进"怪物"的C位
- **A**：在主角心中的最初"理想恋人C"
- **B**：被主角暴力地当成新的C（替代A的位置），因此丧命
- **E**：在B案后主动说"叫她C"，试图成为现实中"为你背负一切的人"

**关键特征**：
- C是概念性的"位置"，不是具体的人物
- 在最终meta环节，E与主角内心创造出的理想化"她"（C）融合，成为跳出游戏框架与玩家对话的存在

## 3. 叙事诡计执行清单

### 3.1 X合一诡计（A与B的混淆）

**核心机制**：让玩家误以为A和B是同一个人，通过避免具体名字、时间模糊、细节嫁接实现。

**执行要点**：
- ✅ **必须做**：
  - 统一使用"她"、"X"、"那个人"等模糊称呼
  - 避免具体的日期标记，只用"那天"、"当时"
  - 将B的日常细节嫁接到A的时间线
  - 在主角的叙事里，第二起事件中"B的消失"被讲述为"X一年前的死亡后续阴影"
  
- ❌ **不能做**：
  - 不能在同一片段中明确区分A和B
  - 不能使用特定人物专属细节（除非是诡计设计的伏笔）
  - 不能明确标注日期，让玩家意识到这是不同年份的事件

**伏笔植入**：
- 细节矛盾：如"喝茶 vs 喝可乐"、"护手霜 vs 汽水"
- 性格差异的微妙暗示（A的理性 vs B的感性）
- 专属物品的错位（主角将A的护手霜放在B的桌上）

**拆分时机**：
- 阶段4（破绽阶段）通过"两问一锁"机制正式拆分
- 玩家需要用`主时间+辅助时间`来区分A案和B案

### 3.2 时间诡计（同一时间，不同事件）

**核心机制**：让玩家误以为同一时间点只对应一个事件，实际上对应多个不同年份的事件。

**执行要点**：
- ✅ **必须做**：
  - 对于承载多个事件的"重叠时间"（如10:37、11:00），采用"两问一锁"机制
  - 第一问：输入"现在几点？" → 玩家输入10:37/11:00
  - 第二问：出现补问，例如"我记得，那天是从几点开始不对劲的？"
  - 玩家再输入一个时间（如约见时间/被通知时间），形成组合键
  - 不同组合对应不同日期与事件：
    - (10:37, X₁) → 一年前A被杀的那天的10:37
    - (10:37, X₂) → 现在B被杀的那天的10:37
  
- ❌ **不能做**：
  - 不能明确标注日期，让玩家意识到这是不同年份的事件
  - 不能在同一时间点直接区分两个事件（必须通过"两问一锁"）

**伏笔植入**：
- 在阶段3（受害者剧本阶段），播放混合版死亡场景（A的坠落+B的对峙）
- 声音和画面的错位是最大的破绽

### 3.3 人物混淆诡计（H/D的模糊化）

**核心机制**：前中期将所有次要人物模糊化，让玩家无法清晰区分。

**执行要点**：
- ✅ **必须做**：
  - 前中期：H/D都是"那个人""那位同学""总在旁边说话的人"
  - 通过"标签先行"给H贴上负面标签（"那个暴躁的家伙"、"总打架的H"）
  - D被模糊成一个无名的、功能性的"朋友"角色
  
- ❌ **不能做**：
  - 不能在前中期明确区分H和D的身份
  - 不能过早揭示H的善意动机

**伏笔植入**：
- H的善意提醒（告诉A关系不健康）与负面标签形成矛盾
- D的"菠萝包"证据成为后期质疑主角的关键

### 3.4 C+E穿插诡计（理想她与共犯的混淆）

**核心机制**：让玩家以为"理想她C"和"现实她E"都是同一个模糊人。

**执行要点**：
- ✅ **必须做**：
  - 在关键节点插入与"理想她C"的白房间对话
  - 在关键节点插入与E的暧昧对话
  - 让玩家以为"理想她 + 现实她 + 过去她"都是同一个模糊人
  - 用E作为"旁观者/共犯"，慢慢透露：其实有**两次**、**两个人**
  
- ❌ **不能做**：
  - 不能过早明确区分C（理想她）和E（现实共犯）
  - 不能直接揭示E的共犯身份

**伏笔植入**：
- E的视角片段（日记、内心挣扎）暗示她知道一些事情
- C的声音从温柔到严厉的变化，暗示"理想"变成"魔鬼"

### 3.5 主角自我洗脑诡计

**核心机制**：主角在无意识中，用A的模板覆盖B的现实，把B替换为A。

**执行要点**：
- ✅ **必须做**：
  - 主角将内心理想化的"她"（由A与罪恶感混合而成）投射在B身上
  - 在日常相处中：用对A的话模板套在B身上
  - 将A的兴趣误投给B（喜欢喝茶 vs 喜欢汽水）等，出现记忆错位的小破绽
  - 主角的内心戏全是如何"赢回"她，而不是反思自己的问题
  
- ❌ **不能做**：
  - 不能直接揭示主角的自我洗脑机制
  - 不能过早暴露记忆错位

**伏笔植入**：
- 细节矛盾：如"喝茶 vs 喝可乐"
- 主角下意识说出与A相关的习惯，随即改口
- D的反对和主角的坚持，是第一个强烈的**矛盾线索**

### 3.6 非线性探索诡计

**核心机制**：设计一批"一段里带两、三个矛盾线索"的切片，玩家不会线性顺序刷完，而是在矛盾中来回跳转，自己组装真实时间线。

**执行要点**：
- ✅ **必须做**：
  - 设计"一段里带两、三个矛盾线索"的切片
  - 玩家不会线性顺序刷完，而是在矛盾中来回跳转
  - 首次引入非主角视角，打破叙事常规（如B的日记、D的视角）
  
- ❌ **不能做**：
  - 不能过早揭示所有矛盾
  - 不能直接给出完整的时间线

**伏笔植入**：
- D的质问与主流的"H是凶手"叙事形成强烈冲突
- 玩家第一次意识到，"失踪"和"死亡"可能是两件事

### 3.7 虚假选择与推责诡计

**核心机制**：在回忆第二次杀人（B案）时，游戏会提供看似有意义的选项，让玩家误以为自己做出了"杀死B"的决定，从而产生"是我选择了暴力"的错觉。

**执行要点**：
- ✅ **必须做**：
  - 在回忆第二次杀人（B案）时，游戏会提供看似有意义的选项
  - 无论玩家选择什么，结果都是B被杀
  - 这个选择是"我"（主角）在回忆中重构的虚假记忆，目的是将责任转嫁给玩家
  - 在虚假选择之后，"我"会直接对玩家说话（打破第四面墙）
  - 通过嘲讽将整件事情的责任推给玩家："你看，是你选的。""是你让她死的。""不是我，是你。"
  
- ❌ **不能做**：
  - 不能过早揭示这是虚假选择
  - 不能直接告诉玩家这是主角的推责

**伏笔植入**：
- 主角的推责机制在前期就有体现（如栽赃给H）
- 主角对玩家的"觉察"与"反抗"逐渐增强

## 4. 格式规范清单

### 4.1 YAML Front Matter

- [ ] 片段ID：格式为 `P阶段-类型-T时间-名称`（如 `P1-DTL-T0805-Breakfast`）
- [ ] 时间：格式为 `HH:MM`（如 `08:05`）
- [ ] 阶段：`P0` 至 `P6`
- [ ] 解锁条件：Flag名称（如 `FLAG_Game_Start`），多个标记用逗号分隔
- [ ] 描述：简短描述（用于开发调试）
- [ ] 作用：片段在游戏中的作用（用于理解创作意图）

**可选字段**：
- 诡计执行：说明该片段执行的叙事诡计
- 伏笔：说明该片段埋下的伏笔
- 给予的线索：说明该片段给予玩家的线索

```yaml

片段ID: P1-DTL-T0805-Breakfast

---
```

### 4.2 文本标记

**内心独白/旁白**：
- [ ] 使用 `*斜体*` 标记
- [ ] 这是主角的内心想法、回忆、分析、环境描写等
- [ ] 对应程序格式的 `narration` 类型

**强调**：
- [ ] 使用 `**粗体**` 标记
- [ ] 用于关键念头、重要信息、时间等需要强调的内容
- [ ] 技术美术会根据情节判断如何转换（bold、red、typewriter等）

- [ ] 主角的对话直接写，无角色名
- [ ] 其他角色对话格式：`角色名: "对话内容"`
- [ ] 对应程序格式的 `dialogue` 类型

**换行**：
- [ ] 自然换行即可，不需要特殊标记
- [ ] 换行表示节奏停顿，技术美术会根据情节判断如何转换
- [ ] 可能需要 `{br}`、`{delay}` 等技术标签，但这是技术美术的工作
