#### **第一部分：“人间失格”氛围的四大支柱 (写作原则)**

**原则一：不可靠的第一人称——“我的辩解”**
太宰治的主角总是在“辩解”，向一个不存在的法官或读者解释“我为何如此”。他说的未必是谎言，但永远是经过自我美化和扭曲的“他的真实”。
*   **执行标准**:
    *   **多用自问自答**: “我为何要那么做呢？大概是因为…”、“旁人看来或许很奇怪吧。但我只是…”
    *   **淡化恶意，强调“懦弱”与“恐惧”**: 主角在叙述自己的恶行时，不会说“我出于恶意”，而会说“我因为害怕失去，才做了那件无可奈何的事。”
    *   **对他人行为的病态解读**: 将B的觉醒解读为“她不再纯粹了”，将D的关心解读为“多管闲事的窥探”。

**原则二：疏离感的营造——“我与世界之间隔着一层薄膜”**
主角无法真正融入世界。他的一切行为都是在“扮演”一个正常人。他对周围的一切都有一种抽离的、冷漠的观察感。
*   **执行标准**:
    *   **感官描写的“失焦”**: 描写环境时，多用模糊、失焦的词语。不说“阳光明媚”，而说“某种刺眼的光线从窗外渗进来，让我感到一阵眩晕。”
    *   **将情感“物体化”**: 不直接说“我很难过”，而说“一种类似悲伤的、沉重的东西，盘踞在我的胸口。”
    *   **引用他人的“正常”**: 频繁引用他人的话语或行为，来反衬自己的格格不入。“‘普通人’在这种时候，似乎应该会哭吧。但我一滴眼泪也流不出来。”

**原则三：耻感与自我厌恶——“我是个怪物”**
尽管主角不断辩解，但在其内心深处，是对自己存在的强烈“耻感”和厌恶。这种情绪不是以外显的忏悔，而是以自嘲、自我贬低的形式出现。
*   **执行标准**:
    *   **突兀的自我嘲讽**: 在一段看似正常的叙述后，突然插入一句自我否定的内心独白。例如，在描写完自己如何体贴B之后，内心独白：“（看呐，一个多么完美的、体贴的怪物。）”
    *   **对赞美的恐惧**: 当被人夸奖时，主角的内心反应不是喜悦，而是恐慌，因为“他们称赞的那个‘我’，根本不存在。”

**原则四：语言的“表演性”——精致的颓废**
太宰治的文风是颓废的，但绝不粗糙。它有一种经过精心打磨的、病态的美感。用词讲究，句式从容，即使在描述最不堪的情感时，也保持着一种奇异的优雅。
*   **执行标准**:
    *   **避免口语化、网络化的词语**: 除非是引用他人对话，否则主角的内心独白应保持书面语的精致。
    *   **多用比喻，尤其是自然或物理比喻**: “我的谎言像藤蔓一样，缠绕住她，也缠绕住我自己。”、“这份爱，像一块冰，被我死死攥在手心，既不愿它融化，又被冻得生疼。”

#### **第二部分：“黄金切片”的五维质量模型 (审查标准)**

*对于每一个切片，都必须用以下五个维度进行拷问，全部通过才算合格。*

**维度一：【逻辑功能】(Functionality)**
*   **检查清单**:
    *   [ ] **线索清晰吗？**: `给予的线索` 是否明确指向一个或多个可探索的时间点/关键词？玩家看完后知道下一步该干嘛吗？
    *   [ ] **条件合理吗？**: `进入条件` 是否与其前置切片逻辑自洽？这个Flag真的能保证玩家在理解当前剧情后才进入此片段吗？
    *   [ ] **Flag设置正确吗？**: `设置的Flag` 是否准确地概括了玩家在本片段获得的核心新认知？

**维度二：【诡计贡献】(Deception)**
*   **检查清单**:
    *   [ ] **服务于哪个诡IT计？**: `关联诡计` 字段是否明确？这段文本是如何具体执行这个诡计的？
    *   [ ] **误导性足够吗？**: （对于前期切片）这段描述是否巧妙地引导玩家走向错误的结论？（例如，将A的特征和B的日常无缝拼接）
    *   [ ] **破绽是否可发现？**: （对于中后期切片）其中包含的矛盾点（破绽），是否对足够细心的玩家来说是可见的？是否“公平”？

**维度三：【情感锚点】(Emotion)**
*   **检查清单**:
    *   [ ] **符合当前阶段目标吗？**: 这段文字引起的情感（甜蜜/不安/悲伤/怀疑/恐惧）是否符合七阶段大纲对当前阶段的情感定义？
    *   [ ] **有记忆点吗？**: 片段中是否存在一个强烈的、令人印象深刻的意象、对话或感受？（例如，“可乐滚落的声音”、“练习悲伤的表情”）
    *   [ ] **主角弧光一致吗？**: 主角在此刻的内心活动，是否符合他从“病态控制”到“彻底暴露”的整体心理变化轨迹？

**维度四：【文体风格】(Style)**
*   **检查清单**:
    *   [ ] **符合“四大支柱”吗？**: 这段文字是否体现了“不可靠的第一人称”、“疏离感”、“耻感”和“语言的表演性”中的至少两种？
    *   [ ] **是“展示”而非“告知”吗？**: 是否避免了用形容词直接告知读者主角的感受（如“我很难过”），而是通过行为和内心独白来展示？
    *   [ ] **文本有节奏感吗？**: 长短句结合是否得当？关键信息是否放在了最有力度的位置？

**维度五：【互动实现】(Interaction)**
*   **检查清单**:
    *   [ ] **注释是否有效？**: 特殊设计点是否已用`//注释`标出，解释其背后的设计意图？
    *   [ ] **Tag是否丰富演出？**:
        *   `{red}{/red}` 是否在关键时刻增强了氛围？
        *   `{delay:1.2}` 是否在主角紧张时或者强调时控制文本节奏？
        *   `{strike}{/strike}` 是否在播放被篡改的记忆时，让文本出现划线修正？
    *   [ ] **选项是否有意义？**:
        *   **虚假选择**: 选项是否体现了玩家意图与主角行为的冲突？（玩家选A，主角却做了B，并给出内心独白解释“我为什么不能那么做”）
        *   **认知选择**: 选项是否用于确认玩家的理解？（例如，在后期提问“杀死她的是谁？”，选项有H和“我”，选错会有特殊反馈）
