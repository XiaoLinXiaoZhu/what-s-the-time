## 核心理念

### 一、文风的目标

本游戏的文风旨在建立"人间失格"氛围，其最终体验并非解谜的快感或情感的宣泄，而是类似于读完《人间失格》后，那种对人性深渊的凝视所带来的、无法言说且挥之不去的压抑与虚无感。

**核心特征**：
- **拒绝提供简单的道德答案**：不进行说教，不提供审判，不寻求意义
- **通过文本本身的力量**：让玩家体验从"甜蜜日常"到"发现真相"再到"自我怀疑"的情感历程
- **最终落点**：无法言说、挥之不去的压抑、虚无和对人性深渊的凝视

### 二、文风在游戏叙事中的作用

文风服务于游戏的四个情感阶段，每个阶段都有明确的情感目标：

**前期**：让玩家代入主角，体验类似于校园恋爱题材的作品，但带有病态控制的色彩。玩家与主角的情感锚点是"和B的情感"，但这种情感从一开始就带有病态控制的色彩——玩家代入的是一个看似深情、实则正在将恋人"物品化"的角色的视角。文风需要营造"甜蜜中的诡异"，让玩家在享受甜蜜的同时，隐约会感到一丝不协调和控制感。

**中期**：让玩家与主角共情"B怎么死了？我是受害者"这样的共情路径。玩家需要扮演主角证明自己的无罪和伤痛。文风需要强化主角的"受害者形象"，让玩家相信主角是无辜的。

**中后期**：让玩家逐渐发现主角的异常，跟随这个异常重新探索新的时间片段以及整理整个故事的真实脉络。玩家的情绪从"我爱的人怎么死了？"转变为"我们（我和主角）的爱，是不是从一开始就错了？"这种自我怀疑，这种对自己前期情感投入的否定，是通往"人间失格"式虚无感的入口。文风需要逐步揭示主角的真实面貌，让玩家意识到"创伤后遗症"只是面具。

**后期**：meta元素让玩家扮演主角执行杀人，和C（理想化的"她"）跳出游戏框架，询问主角时间。此时玩家的现在时间和游戏的虚拟时间达成统一。文风需要通过meta元素，将"人间失格"的体验从观察主角，转变为反思自我。玩家在凝视深渊之后，发现深渊也在凝视自己。

### 三、文风的核心特征（四大支柱）

为了实现上述目标，文风需要具备以下四个核心特征：

1. **不可靠的第一人称**：主角的叙述是经过自我美化和扭曲的"他的真实"，他说的未必是谎言，但永远是经过自我美化和扭曲的"他的真实"。他通过"辩解"来维护自己的"受害者形象"，将责任转嫁给他人。

2. **疏离感**：主角无法真正融入世界，他的一切行为都是在"扮演"一个正常人。他对周围的一切都有一种抽离的、冷漠的观察感。这种疏离感让玩家感受到主角与世界的隔阂，也为后续的真相揭露埋下伏笔。

3. **耻感与自我厌恶**：尽管主角不断辩解，但在其内心深处，是对自己存在的强烈"耻感"和厌恶。这种情绪不是以外显的忏悔，而是以自嘲、自我贬低的形式出现。这种内在的冲突，让主角的形象更加复杂和真实。

4. **语言的"表演性"**：文风是颓废的，但绝不粗糙。它有一种经过精心打磨的、病态的美感。用词讲究，句式从容，即使在描述最不堪的情感时，也保持着一种奇异的优雅。这种"表演性"正是主角"完美面具"的体现。

## 写作原则（四大支柱）

### 原则一：不可靠的第一人称——"我的辩解"

太宰治的主角总是在"辩解"，向一个不存在的法官或读者解释"我为何如此"。他说的未必是谎言，但永远是经过自我美化和扭曲的"他的真实"。

**执行标准**：
- **多用自问自答**： "我为何要那么做呢？大概是因为…"、"旁人看来或许很奇怪吧。但我只是…"
- **淡化恶意，强调"懦弱"与"恐惧"**：主角在叙述自己的恶行时，不会说"我出于恶意"，而会说"我因为害怕失去，才做了那件无可奈何的事。"
- **对他人行为的病态解读**：将B的觉醒解读为"她不再纯粹了"，将D的关心解读为"多管闲事的窥探"。

**创作技巧**：
- 在主角的内心独白中，频繁使用"大概"、"或许"、"似乎"等不确定词汇
- 让主角为自己的行为寻找"合理"的解释，即使这些解释在读者看来是扭曲的
- 通过对比"旁人看来"和"我的真实"，突出主角的不可靠性

### 原则二：疏离感的营造——"我与世界之间隔着一层薄膜"

主角无法真正融入世界。他的一切行为都是在"扮演"一个正常人。他对周围的一切都有一种抽离的、冷漠的观察感。

**执行标准**：
- **感官描写的"失焦"**：描写环境时，多用模糊、失焦的词语。不说"阳光明媚"，而说"某种刺眼的光线从窗外渗进来，让我感到一阵眩晕。"
- **将情感"物体化"**：不直接说"我很难过"，而说"一种类似悲伤的、沉重的东西，盘踞在我的胸口。"
- **引用他人的"正常"**：频繁引用他人的话语或行为，来反衬自己的格格不入。"'普通人'在这种时候，似乎应该会哭吧。但我一滴眼泪也流不出来。"

**创作技巧**：
- 使用"某种"、"类似"、"仿佛"等模糊化词汇
- 将抽象情感具象化为物体或物理现象
- 通过对比"他们"和"我"，突出主角的异质性

### 原则三：耻感与自我厌恶——"我是个怪物"

尽管主角不断辩解，但在其内心深处，是对自己存在的强烈"耻感"和厌恶。这种情绪不是以外显的忏悔，而是以自嘲、自我贬低的形式出现。

**执行标准**：
- **突兀的自我嘲讽**：在一段看似正常的叙述后，突然插入一句自我否定的内心独白。例如，在描写完自己如何体贴B之后，内心独白："（看呐，一个多么完美的、体贴的男友。）"
- **对赞美的恐惧**：当被人夸奖时，主角的内心反应不是喜悦，而是恐慌，因为"他们称赞的那个'我'，根本不存在。"

**创作技巧**：
- 在看似正常的叙述后，突然插入括号内的自我嘲讽
- 让主角对正面评价产生怀疑和恐惧
- 通过自我贬低，揭示主角内心的真实状态

### 原则四：语言的"表演性"——精致的颓废

太宰治的文风是颓废的，但绝不粗糙。它有一种经过精心打磨的、病态的美感。用词讲究，句式从容，即使在描述最不堪的情感时，也保持着一种奇异的优雅。

**执行标准**：
- **避免口语化、网络化的词语**：除非是引用他人对话，否则主角的内心独白应保持书面语的精致。
- **多用比喻，尤其是自然或物理比喻**： "我的谎言像藤蔓一样，缠绕住她，也缠绕住我自己。"、"这份爱，像一块冰，被我死死攥在手心，既不愿它融化，又被冻得生疼。"

**创作技巧**：
- 使用书面语，避免网络用语和口语化表达
- 多用比喻，尤其是自然现象和物理现象的比喻
- 即使在描述最不堪的内容时，也要保持语言的优雅

## 表现技法

### 技法一：反复/纠结/对比表现

**原则**：分行显示才有张力

当主角在两个念头之间徘徊、反复追问、对比矛盾时，必须让句子分行显示，形成上下对照的视觉节奏。

**适用场景**：
- 自我追问：`时间不对。哪里不对？`
- 对比怀疑：`现在是……这个时间吗？还是说，那只是某一次……坠落的时间？`
- 递进强调：`只有风声，只有红色，只有……`
- 反复尝试：`我试图拼凑，试图抓住，但只有虚无。`

**创作技巧**：
- 将纠结的念头分行，形成视觉上的对比
- 使用省略号营造不确定感
- 通过重复结构（"只有...只有...只有..."）强化情绪

### 技法二：比喻与意象策略

**原则**：画面感优先于概念

**错误示例**（概念先行，画面感弱）：
```
模糊的红色。
像颜料一样，在视野里晕开...
```
"模糊的红色"过于抽象，玩家没有画面感。

**正确示例**（比喻前置，画面感强）：
```
模糊的红色，像颜料一样。
在视野里晕开，蔓延，吞噬一切。
```
立刻给玩家"颜料"的具象画面，然后展开动作。

**创作技巧**：
- 将比喻前置，先给出具象画面
- 使用自然现象和物理现象的比喻
- 通过动作词（"晕开"、"蔓延"、"吞噬"）强化画面感

### 技法三：精简与节奏的平衡

**原则**：去除冗余，保留必要的呼吸

叙事应该流畅有张力，应该按照叙事的节奏一张一弛。

**需要精简的**：
- 重复的追问（三个变两个）
- 冗余的形容词（"那个时间，那个瞬间" → "那个瞬间"）

**需要保留的**：
- 反复强调的核心词（"时间不对"出现两次，第二次用强调）
- 递进的"只有"结构（风声/红色/...）
- 关键情绪节点的停顿

**创作技巧**：
- 删除冗余词汇，保留核心信息
- 在关键情绪节点使用自然停顿（通过换行实现）
- 通过反复强调核心词，强化主题

## 创作格式

### 一、基本格式

你只需要像写小说一样创作，使用最自然的格式：

1. **内心独白/旁白**：使用 `*斜体*` 标记
   - 这是主角的内心想法、回忆、分析、环境描写等
   - 例如：`*早晨的阳光透过梧桐叶片的缝隙，像某种精致的花边装饰。*`

2. **强调**：使用 `**粗体**` 标记
   - 用于关键念头、重要信息、时间等需要强调的内容
   - 例如：`*我看着她的背影，想起了一个问题。**时间不对。**哪里不对？*`

3. **对话**：使用自然格式 `角色名: "对话内容"`
   - 主角的对话不需要角色名，直接写对话内容
   - 其他角色的对话需要标注角色名（如 `B: "..."`、`C: "..."`）
   - 例如：
     ```
     她指着街角那家新开的包子店，眼睛亮晶晶的。"我们去试试那个吧？听说皮薄馅大，很香的。"
     
     B: "你刚才有没有看到？我们班那个H，刚才在走廊里……"
     ```

4. **换行**：自然换行即可，不需要特殊标记
   - 换行表示节奏停顿，技术美术会根据情节判断如何转换

### 二、片段元数据

在文件开头使用 YAML Front Matter 格式标记片段信息：

```markdown
---
片段ID: P1-DTL-T0805-Breakfast
时间: 08:05
阶段: P1
解锁条件: FLAG_Game_Start
描述: 阶段1 - 早餐
作用: 建立"虚构恋爱史"的开端；情感绑定。
---
```

**字段说明**：
- `片段ID`：片段的唯一标识符（格式：P阶段-类型-时间-名称）
- `时间`：片段对应的时间点（HH:MM格式）
- `阶段`：片段所属的游戏阶段（P0-P6）
- `解锁条件`：解锁该片段所需的标记（多个标记用逗号分隔）
- `描述`：片段的简短描述（用于开发调试）
- `作用`：片段在游戏中的作用（用于理解创作意图）

### 三、重要提示

**你只需要专注于创作**：
- ✅ 像写小说一样自然写作
- ✅ 使用 `*斜体*` 表示内心独白/旁白
- ✅ 使用 `**粗体**` 表示强调
- ✅ 使用自然对话格式 `角色名: "对话内容"`
- ✅ 自然换行，不需要关心技术细节

**技术美术会处理**：
- ❌ 不需要理解技术标签（`{br}`, `{delay}`等）
- ❌ 不需要理解程序结构（narration vs dialogue）
- ❌ 不需要关心节奏控制（技术美术会根据情节判断）

## 创作示例

### 完整示例

```markdown
---
片段ID: P1-DTL-T0805-Breakfast
时间: 08:05
阶段: P1
解锁条件: FLAG_Game_Start
描述: 阶段1 - 早餐
作用: 建立"虚构恋爱史"的开端；情感绑定。
---

*早晨的阳光透过梧桐叶片的缝隙，像某种精致的花边装饰。*

*这是新学校的第一周。一切都应该是完美的。*

她指着街角那家新开的包子店，眼睛亮晶晶的。"我们去试试那个吧？听说皮薄馅大，很香的。"

我看着她，心里涌起一种奇怪的情绪。不是爱——那种东西太虚假了。

是一种……占有欲？不对。是保护欲？也不对。

"你的胃不好，早上吃这么油腻的东西会不舒服的。"

B: "她眨了眨眼，像是被什么小东西绊了一下。"可是……偶尔试试也没关系吧？"

"偶尔？"我笑了笑，去旁边的便利店买了三明治。

"你看，这个蛋白含量高，营养更均衡。"

她接过三明治，咬了一口，然后露出一个标准化的笑容。"嗯，很好吃。"

*那种笑容。*

*我见过。在很久很久以前，另一个女孩也露出过这样的笑容。*

*那种被训练出来的、迎合式的笑容。*

"中午的时候，我们还是去那个老地方吧。"她走在前面，声音里没有任何起伏。

"还是老地方。"我重复着，仿佛在确认什么。

*我看着她的背影，想起了一个问题。*

***她今天露出的疲惫笑容，是从哪里学来的？***
```

### 创作要点分析

1. **四大支柱的体现**：
   - **不可靠的第一人称**：主角为自己的控制行为寻找"合理"解释（"你的胃不好"）
   - **疏离感**：使用"某种"、"像某种"等模糊化词汇
   - **耻感与自我厌恶**：通过"那种被训练出来的、迎合式的笑容"暗示自我认知
   - **语言的"表演性"**：使用比喻（"像某种精致的花边装饰"），保持书面语的精致

2. **表现技法的运用**：
   - **反复/纠结**：通过分行，表现主角内心的纠结（"占有欲？不对。是保护欲？也不对。"）
   - **比喻与意象**：使用具象比喻（"像某种精致的花边装饰"、"像某种小东西绊了一下"）
   - **精简与节奏**：去除冗余，保留核心信息，通过换行实现节奏控制

3. **格式的使用**：
   - `*斜体*` 用于内心独白/旁白
   - `**粗体**` 用于强调（如最后的问题）
   - 自然对话格式，主角对话不需要角色名
   - 自然换行，技术美术会根据情节判断如何转换

## 常见问题

**Q: 如何区分内心独白和说出口的话？**  
A: 如果这句话是"脑内声音"，使用 `*斜体*`；如果"有声音"（哪怕是压抑的、小声的），直接写对话内容。

**Q: 时间信息需要特殊标记吗？**  
A: 不需要。你只需要在文本中自然提及时间，技术美术会根据情节判断是否需要特殊处理。

**Q: 如何表示强调？**  
A: 使用 `**粗体**` 标记即可。技术美术会根据情节判断是否需要使用特殊效果（如逐字显示、红色等）。

**Q: 换行有什么要求吗？**  
A: 自然换行即可。换行表示节奏停顿，技术美术会根据情节判断如何转换（是否需要delay、是否需要拆分成独立item等）。

**Q: 选择分支怎么写？**  
A: 像写小说一样，自然描述选择场景。技术美术会根据情节判断如何实现选择分支。
