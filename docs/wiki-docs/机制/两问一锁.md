# 两问一锁

## 概念

一种信息锁机制。某些深层真相需要【玩家】在多个周目中获取两个关键时间点/信息才能解锁。

## 示例

**【A案】真相版（P01-04）**的解锁条件：
- 第一问：【玩家】命中时间 10:37（触发假闪回版 P01-03）
- 第二问：【玩家】命中时间 08:20（【A】发约楼顶的消息）
- 两者都满足后，再次进入 10:37 时，解锁完整动作版——看到【主角】有意识地"多推了一点点"

## 设计目的

- 避免【玩家】一次性看到全部真相
- 通过多周目的信息积累逐步拆开谜底
- 让"假闪回"和"真相版"形成强烈对照
- 密码锁、电脑密码等也是需要通过多周目获取线索才能开启的"信息锁"
