# 代词模糊

## 定义

游戏中故意不明确"他"、"她"、"它"等代词的指代对象，让【玩家】需要结合线索自行推理。

## 核心应用

- "她"在全游戏中长期不区分是【A】还是【B】（因为同名 X），直到真相周目才被一次性拆分
- 【主角】使用模糊表达：将主动变为被动（"她从那里掉下去了"而不是"我把她推下去了"）
- 【G】在问话中大量使用模糊代词："那件事""她""那时候"，让【A案】与【B案】在语言层面缝合

## 设计目的

- 利用同名（X）制造混淆，让【玩家】无法分辨两个案件
- 增强多周目的信息拼图感
- 配合【叙事诡计】中的"时间诡计"，让同一个时间点（如【10:37】）的"她"变得不可确定
