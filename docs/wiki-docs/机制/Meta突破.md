# Meta突破

## 定义

游戏在终局阶段打破第四面墙，将"叙事时间/游戏时间/现实时间"折叠到一起。

## 触发

在最后的关键周目（【Loop系统】的 Loop G），"她"（由【E】与理想化投射融合的【C位】）第一次明显打破第四面墙：

1. 要求【玩家】输入**现实中的"现在时间"**
2. 若【玩家】输入错误，她报出系统当前时间："不对，现在是 XX:XX，对吧？"
3. 将【玩家】拖入"你在重播这场暴力"的责任中

## 最终结局

与【玩家】对话的是【C位】——【主角】一切罪行的根源和最终目的。她说：

> "你看，他失败了。他不够纯粹，不够坚定，所以他弄脏了一切。但他追求我的心，是真的。那种想要留住永恒、拒绝任何瑕疵的渴望……你，屏幕前的你……你难道没有一瞬间，理解他吗？……手表还给你。现在，轮到你了。What's the time?"

游戏回到最初的输入框。但这一次【玩家】知道，输入的不再是"过去的时间"，而是**自己的、新的"【10:37】"**。

## 结局内核

- 不审判【玩家】的"观看行为"，而是拷问内心是否也存在"控制欲"和"完美主义的魔鬼"
- 不提供任何答案或解脱
- 将"人间失格"的体验从观察【主角】变成反思自我
- 【玩家】在凝视深渊之后，发现深渊向自己递出了一块怀表
