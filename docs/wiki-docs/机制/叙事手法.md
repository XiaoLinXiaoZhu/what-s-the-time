# 叙事手法

本作整体模仿 David Lynch 的叙事风格，核心手法如下。

## 不可靠叙述者

【主角】是唯一的视角来源，但他的记忆经过"剪辑"。所有【玩家】看到的场景都是【主角】愿意面对的版本。同一时间点在不同周目下会展示假闪回（模糊版）和真相版。

## 同名混淆

【A】和【B】的名字都是 X，【E】的名字不是 X。游戏中"她"长期不区分指的是谁，直到真相周目才被一次性拆分。配合同一时间点（如 10:37 同时是【A案】和【B案】的核心时刻），让【玩家】无法分辨看到的是哪件事。

## 代词模糊

故意不明确"他""她""它"的指代对象。【主角】使用被动句式回避责任（"她从那里掉下去了"而不是"我把她推下去了"）；【G】在问话中大量使用"那件事""她""那时候"，让两案在语言层面缝合。

## 虚假选择

回忆【B案】时，游戏提供看似有意义的选项。无论【玩家】选什么，结果都是【B】被杀——这是【主角】在回忆中重构的虚假记忆，目的是将责任转嫁。选择后【主角】打破第四面墙嘲讽推责："你看，是你选的。"

## 打破第四面墙（Meta 终局）

最终周目中，"她"（由【E】与理想化投射融合的【C位】）要求【玩家】输入**现实中的当前时间**。若输入错误，她报出系统时间："不对，现在是 XX:XX，对吧？"

这一步将叙事时间、游戏时间、现实时间折叠到一起，把【玩家】拖入"你在重播这场暴力"的责任中。游戏回到最初的输入框，变成一个开放的循环——【玩家】输入的不再是"过去的时间"，而是自己的、新的"10:37"。

结局不审判【玩家】的观看行为，而是拷问内心是否也存在"控制欲"和"完美主义的魔鬼"。不提供任何答案或解脱。
