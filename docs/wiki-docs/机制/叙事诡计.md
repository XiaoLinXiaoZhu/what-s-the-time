# 叙事诡计

整体模仿 David Lynch 的叙事风格。核心设计抓手如下：

## 角色的自知/玩家不知

【主角】或许隐藏大量真相和设定值得被挖掘。

- **基础层**：【主角】会说"我应该记得密码"，但【玩家】并不知道 → 产生角色不会主动打开密码箱的"错觉"
- **外部塑造层**：【玩家】需要通过外部信息了解"我是谁"。外部评价不一定真实，可能片面、误导，甚至有栽赃（参见【H】的遭遇）
- **深层身份**：【玩家】到底操控的是谁？真的是他人口中称呼的那个"我"吗？

## 【代词模糊】

"他"、"她"、"它"到底指代谁？游戏中"她"长期不区分是【A】还是【B】（因为同名 X），直到真相周目才被拆分。

## 时间诡计

- 【A案】和【B案】都发生在【10:37】，【玩家】无法区分看到的是哪个 10:37
- 【主角】的记忆被剪辑过，不同周目展示同一时间点的不同版本

## 【虚假选择】

回忆【B案】时，游戏提供看似有意义的选项。无论【玩家】选什么，结果都是【B】被杀。这是【主角】在回忆中重构的虚假记忆，目的是将责任转嫁给【玩家】。

选择后【主角】会打破第四面墙嘲讽推责："你看，是你选的。""是你让她死的。"

## 【Meta突破】

在最后的关键周目，"她"第一次明显打破第四面墙，要求【玩家】输入现实中的"现在时间"。若输入错误，她报出系统当前时间。

这一步将"叙事时间/游戏时间/现实时间"折叠到一起，把【玩家】拖入责任链。

## 名字诡计

【A】和【B】的名字都是 X，【E】的名字不是 X。游戏利用同名制造混淆，让【玩家】无法区分"她"指的是谁。
