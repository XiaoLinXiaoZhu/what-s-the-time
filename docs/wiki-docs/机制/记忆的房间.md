# 记忆的房间

## 核心设计哲学

> 真实时间线不是起点，它是终点。它是谜底。

【玩家】体验不是一条线，而是一个房间——一个【主角】记忆构成的房间。墙上挂着很多时钟，大部分都停了。【玩家】可以拨动任何一个时钟的指针，通过墙上的小孔窥看那个时间点发生的事。

**你的工作是【建筑师不是导游】。**

## 房间设计四要素

每个关键【时间锚点】就是一个房间，需要标注四个维度：

1. **谎言**：这个房间想让人相信什么？
2. **真相**：实际发生了什么？
3. **初次气味**：【玩家】第一次进入时的感受（甜蜜？怀疑？悲伤？）
4. **重访发现**：第十次进入时会发现什么新东西？

## 示例

### 【03:15】：童年的房间
- **谎言**："这只是普通的家庭生活。"
- **真相**："控制与恐惧的种子已经种下。"
- **初次感受**：压抑、不安
- **重访发现**：【主角】在家庭冲突中学会的"表演"技巧

### 【08:05】：早餐的房间
- **谎言**："我只是在关心她的健康。"
- **真相**："控制已经开始了。"
- **初次感受**：甜蜜、日常
- **重访发现**：【A】笑容中的疲惫，【主角】替她做决定时的坚决

### 【10:37】（A案）：旧天台的房间
- **谎言**："这是一场意外。"
- **真相**："这是一场预谋。"
- **初次感受**：震惊、悲痛
- **重访发现**：【主角】推【A】时那个"顺势"的动作有多精准

## 与【Loop系统】的关系

每个 Loop 为【玩家】提供一组"明显被提示的时间"加一些"自由撞出来的时间"。房间内容可能根据 Loop 阶段和已解锁的 flag 展示不同版本。
