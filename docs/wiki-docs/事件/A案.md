# A案（一年前的旧案）

## 概述

一年前，【主角】在校园楼顶于【10:37】有计划地杀死恋人【A】，并通过精心设计的陷阱将罪名栽赃给【H】。

## 时间线

### 案前

- **长期**：【主角】以"关心"为名控制【A】的一切
- **【08:05】**：两人一起上学，【主角】替【A】决定早餐（控制的日常体现）
- **【12:10】**：【A】暗示"最近有点累""我们找个时间聊聊"（关系问题早期警讯）
- **【H】的善意提醒**：【H】告诉【A】这段关系不健康，被【主角】偷听到

### 案发当天

- **08:20**：【A】发消息约楼顶"聊聊"，【主角】视为"机会"
- **10:05**：两人到楼顶，聊着看似普通的话题，气氛微妙
- **【10:37】**：
  - 【A】提出分手/揭露秘密
  - 【主角】在"想拉住她"的伪装动作中发力将她推下
  - 【H】因【主角】设计的陷阱（纸条诱导 + 栏杆松动）赶到天台，在试图保护【A】时无意中触碰到她
- **10:08-10:10**：【主角】疯狂看表，确认时间没有倒流；开始呼救、假装慌乱；冷静调整现场细节

### 善后

- **11:00**：学校问话，【主角】开始有意识地将矛头引向【H】
- 通过栽赃策略（参见【H】的"被栽赃过程"），将社会叙事中的【C位】推给【H】
- 最终结论：替罪羊【H】被判定为主要责任人

### 远视目击

- **16:10**：【E】从远处看到楼顶剪影和"不正常的动作"，选择低头装作没看见

## 游戏中的呈现

- **假闪回版**（P01-03，首次命中10:37）：模糊版，只见踉跄和坠落，【主角】自述"我没拉住她"
- **真相版**（P01-04，满足【两问一锁】后）：完整动作版，与假闪回形成强烈对照
