# B案（现在的案件）

## 概述

【主角】升学后遇到与【A】同名（X）的【B】，将理想化人格投射于她。当【B】偏离"理想模板"时，在相同的心理驱动下再次杀人。

## 时间线

### 案前（甜蜜中的病态）

- 【主角】和【B】的日常看起来是校园恋爱喜剧
- 用【B】的生日作为时间锚点，包裹准备生日礼物等日常故事
- 【E】在此期间扮演"共鸣者"，附和【主角】对【B】的不满
- 【D】作为【B】的朋友，逐渐察觉异常

### 案发当天

- **【10:37】（B案）**：【B】与【主角】对峙，被杀
- **【10:50】**：【E】到场，明确目击全过程，选择成为共犯
  - 对【主角】说"叫她 C""她会帮他处理掉 C"
  - 协助处理尸体

### 善后

- **11:00（B案）**：问话场景，【G】用几乎和【A案】一样的话问【主角】（形成镜像）
- 校园视角：【B】目前只被视为"失踪/请假"
- 【D】是最早公开指出【主角】问题的人
- 流言再起，【E】被当作"凶手"
