# C位（概念性"位置"）

> C 不是某个具体人物，而是一个"角色位"。

## 定义

C 是一个可以被任何人占据的概念性位置，包含两重含义：

1. **【主角】心中"可以为我做任何事的人"**——理想恋人的位置
2. **社会故事中"那件事里的怪物/关键人物"**——替罪羊的位置

## 占据过 / 试图占据 C位的人

| 角色 | 方式 | 结果 |
|------|------|------|
| 【A】 | 【主角】心中最初的"理想恋人" | 被杀害后成为永恒的投射原型 |
| 【H】 | 被社会故事塞进"怪物"的位置 | 成为【A案】的替罪羊 |
| 【B】 | 被【主角】暴力地当成新的 C（替代【A】） | 因不符合模板而被杀害 |
| 【E】 | 主动说"叫她 C"，试图成为"为你背负一切的人" | 成为共犯，在打破第四面墙中与理想化"她"融合 |

## 叙事功能

- 强化"名字/位置可以被替换，但暴力模式一直在"的主题
- 让【玩家】在多周目中不断追问："现在被当作 C 的，是谁？"
- 揭示社会如何需要一个"怪物"来承担罪责，以及加害者如何利用这一点
