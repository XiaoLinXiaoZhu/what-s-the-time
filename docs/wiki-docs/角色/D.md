# D（B 的朋友）

## 身份

【B】的好友/室友/社团同伴，对【B】日常状态最敏感的人。与【主角】有普通同学关系。

## 核心驱动

"为【B】说话"，而不是"破案"。当所有人说"她可能只是有事回家"，只有 D 坚定认为"她不会不打招呼就消失"。

## 弧光

- **前期**：调侃【主角】和【B】像情侣，帮他们打圆场
- **中期**：私下为【B】担心，逐渐收集矛盾信息，对【主角】态度从中立→怀疑
- **后期**：明确质问【主角】"那天你到底对她说了什么？"，成为【主角】"受害者形象"被瓦解的关键力量

## 叙事功能

- 代表"身边人"视角，展现一个不再相信【主角】叙事的人
- 在多周目中，台词从模糊到笃定
- 代表"拒绝接受虚假叙事"的力量
