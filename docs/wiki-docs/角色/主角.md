# 主角（"我"）

## 身份

游戏中【玩家】操控的角色。【玩家】并不是角色本身，只是在尝试扮演角色。

## 完美面具（表层印象）

- 外在表现：温和、脆弱、善良、孤僻，像"被命运折磨的好孩子"
- 恋爱中表现为：细心、体贴、善解人意，偶尔脆弱得让人心疼
- 在同学和老师（如【F】）面前维持的形象：沉稳、可怜、值得保护
- 在真相揭示前，即便是【玩家】也完全被这个面具所蒙蔽

## 真实面貌

### 悲剧家庭背景

成长于充满控制、暴力和情感虐待的家庭。学会通过"伪装""表演"来保护自己，内化出"要么控制，要么被控制"的思维。

与【03:15】（童年记忆锚点）关联。

### 核心性格特质

- **极致的控制欲**：以"我是在保护你/关心你"为外衣，实际上在替对方做所有决定
- **表演大师**：能在几秒钟内从"失控的杀人犯"切换到"绝望的受害者"
- **自我欺骗**：真正相信了自己编造的谎言，认为自己是受害者
- **物品化倾向**：将恋人视为"可被塑造/替换的对象"，一旦偏离"理想模板"就可能被"废弃"
- **病态的浪漫主义**：对"完美爱情"有近乎宗教般的执念

### 语言与叙事特点

- **永远不说谎，只"选择性陈述"**：所说的话从技术上永远是"事实的子集"
- **用反问代替回答**："你觉得呢？""你认为我会吗？"
- **在关键时刻使用模糊表达**：将主动变为被动（"她从那里掉下去了"而不是"我把她推下去了"）

## 关键事件

### 【A案】相关

- 与【A】交往期间，以关心为名控制她的一切（早餐、社交、学习）
- 当【A】试图分手或揭露秘密时，在校园楼顶预谋杀害她（【10:37】）
- 通过精心设计的陷阱将罪名栽赃给【H】
- 用"受害者剧本"赢得所有人同情

### 【B案】相关

- 升学后遇到与【A】同名的【B】，将理想化人格投射于她
- 当【B】表现出不符合"理想模板"的样子时，在相同的心理驱动下再次杀人
- 【E】在案发后成为共犯

## 叙事功能

- 作为不可靠叙述者，所有记忆片段都经过他的"剪辑"
- 在虚假选择后对【玩家】推责："你看，是你选的。"
- 【玩家】输入时间，实际是在操纵"我"愿意面对的记忆片段
