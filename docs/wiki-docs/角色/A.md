# A（旧案受害者，名字：X）

## 身份

【主角】的前女友，【A案】的受害者。与【B】同名（都叫 X），这是【叙事诡计】的核心之一。

## 性格特质

- **独立、有主见、重视平等与真实**
- 内心善良但不软弱，对不公的事有自己的底线
- 有主见但不好争吵，倾向于用冷处理或理性对话来解决冲突
- 注重隐私，不会随意把感情问题告诉别人

## 与主角的关系

- 初期被【主角】"脆弱、需要被理解"的形象吸引
- 逐渐被【主角】以"关心"之名控制（如替她决定早餐、社交安排等）
- 【08:05】早餐场景是这种控制的典型体现
- 后期想分手/冷静一段时间，触发了【主角】的杀意

## 关键事件

- 【H】察觉关系问题并提醒【A】，被【主角】偷听到
- 在天台与【主角】争执，提出分手/揭露秘密
- 于【10:37】被【主角】推下楼顶身亡

## 叙事功能

- 代表"真实的人"如何被【主角】的理想化投射所取代
- 她的死成为【主角】一切后续行为的原点——【10:37】这个时间锚点
- 与【B】同名（X），让游戏中"她"的代词长期不可区分
