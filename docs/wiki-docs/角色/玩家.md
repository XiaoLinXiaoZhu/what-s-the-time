# 玩家

## 身份

游戏的操作者。在叙事层面被拉入责任链——前中段始终认为自己操控的是"被命运折磨的受害者"，末段才意识到自身也在不断重播暴力。

## 核心机制

- 通过自由输入时间（HH:MM）操纵【主角】愿意面对的【记忆的房间】
- 通过【虚假选择】，误以为自己有控制权，实际只是【主角】的叙事工具

## 情感锚点四阶段

1. **前期**：与【主角】绑定，代入校园恋爱。锚点是"和【B】的情感"
2. **中期**：锚点转为"【B】怎么死了？我是受害者"的共情
3. **中后期**：发现【主角】异常，自我怀疑"我们的爱，是不是从一开始就错了？"
4. **后期**：【Meta突破】将玩家拖入责任链

## 叙事功能

- 代表"观看者"视角，最终被揭示为"共犯"之一
- 通过【虚假选择】体验"似乎是自己决定了暴力"的错觉
- 通过被【主角】推责，感受责任如何被转嫁
- 最终主题：重复观看和操作暴力本身也是一种参与
