# G（警官）

## 身份

一年前【A案】、现在【B案】都参与过的警官。一直记得【主角】这张脸。

## 风格

不夸张、不戏剧化，语气平静但有压迫感。在问话室中大量使用代词模糊："那件事""她""那时候"，让【A案】与【B案】在语言层面缝合。

## 关键事件

- 【A案】：警局初步笔录 + 第二轮问话，【主角】在他面前构建替罪羊形象
- 【B案】：用几乎和旧案一样的话问【主角】
- 尾声：最后一次谈话，对【主角】说"你可以回去了"，【主角】看表仍是【10:37】

## 叙事功能

- 对【玩家】而言是"时间与法律"的代表：你可以躲开记忆，但躲不开他
- 通过重复的问话将过去和现在缝合
