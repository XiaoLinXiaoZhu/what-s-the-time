# E（目击者 + 共犯）

## 身份

- 高中时期与【主角】同校的同学，因喜欢【主角】而选择同一所新学校
- 连接【A案】与【B案】的唯一证人：一年前得知【主角】做了手脚，一年后清晰目击【B案】
- 【B案】后主动成为共犯，自称"C"，试图占据【C位】
- 真实名字**不是 X**（与【A】、【B】不同），这是【叙事诡计】的一部分

## 性格与动机

### 核心驱动：存在主义式的"选择"

E 不是被动的"追随者"，而是有独立哲学的"选择者"：

- 一直是不起眼的"背景板"式人物
- 对【主角】的爱是她生命中第一次感受到如此强烈的情感
- 将参与犯罪视为一种"升华"——摆脱平庸、建立终极连接的仪式
- 她的恶与【主角】的恶质感不同：【主角】是向外的"控制"，E 是向内的"渗透"

### 不是负罪感，是"错失感"

【A案】后她后悔的不是"没有救下【A】"，而是"没有在【主角】最脆弱的时候成为他身边的那个人"。看到【主角】用"受害者剧本"赢得同情，她内心深处是**羡慕**的。

## 弧光（三阶段）

### 前期：沉默的观察者（A案后）

- 偷偷跟踪【主角】，看到他在天台做了手脚（栏杆松动等）
- 没有看到推搡过程，只看到【A】坠落的结果
- 选择沉默，用"可能是错觉"说服自己

### 中期：共鸣者与协助者（B案前）

- 当【主角】抱怨【B】"不像她应该有的样子"时，不去劝解而是**附和**
- 帮【主角】监视【B】、圆谎，扮演"安慰者"和"分析师"
- 不是在破坏他们的关系，而是在**"提纯"**【主角】对理想恋人的需求
- 把自己塑造成"比【B】更懂你"的那个人

### 后期：加冕（B案当天及之后）

- **【10:50】不是"决定"，是"加冕"**：她已经通过了所有考验，终于可以戴上"最后的女主角"的王冠
- 清晰目击【B案】全过程后，出现在现场，戴上手套
- 对【主角】说："她说，叫她 C。""她会帮他处理掉 C。"
- 这里的 C 是高度压缩的符号：既指【A】、也指【B】的尸体、更指围绕 X 这个名字的一切罪与纠缠

### "最后的机会"

E 的手曾伸向口袋，指尖触碰到手机——屏幕上三个数字已经按下：1-1-0。但她看到了【主角】眼中的迷茫和脆弱。手机滑回口袋。她走向了他。

## 与主角的关系

- 从单方面依附发展为互相牵制的恐怖共生
- 后期可能尝试"反向控制"："现在，只有我知道全部的秘密。你离不开我。"
- **最终meta环节**：E 与【主角】内心的理想化"她"融合，成为跳出游戏框架与【玩家】对话的存在

## 叙事功能

- 让故事从"一个人的心理问题"升级为"多人共同成就的怪物系统"
- 展现"沉默的共犯"如何通过不作为和主动参与成为暴力循环的一部分
- 名字诡计的关键：她不是 X，但试图成为"最后的 C"
