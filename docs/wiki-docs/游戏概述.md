# What's the time — 游戏概述

## 类型与愿景

多周目文字冒险。外壳是轻松的校园恋爱喜剧，内核是预谋杀人、记忆扭曲与【叙事手法】。

最终体验不是解谜的快感，而是类似读完《人间失格》后对人性深渊的凝视——无法言说且挥之不去的压抑与虚无感。

## 核心机制

每个周目开头，【主角】都会自问："时间是……？" 【玩家】自由输入时间（HH:MM），游戏据此播放对应的记忆场景。

每个时间输入对应一段【主角】的记忆。同一时间在不同周目、不同解锁条件下可能展示完全不同的版本——从美化过的假闪回到残酷的真相。【玩家】不是在看一条线性故事，而是在一个由记忆构成的空间里，自己选择拨动哪个时钟。

## 核心真相（玩家不可见）

- **两次杀人，同名受害者**：【主角】一年前在校园楼顶于 10:37 杀死恋人【A】；现在又将同名女生【B】视作替代品再次杀害。
- **栽赃与自我受害叙事**：【A案】中将责任栽赃给【H】；【B案】中复用同一套"故事模板"。
- **"她"的模糊身份**：杀死【A】后，【主角】在内心形成理想化的"她"（即【C位】），后投射在【B】身上。
- **时间与记忆的扭曲**：真正卡住【主角】的是第一次杀人那一刻的 10:37。多周目是他反复重播、剪辑记忆的过程。

## 体验节奏

- **前期**：拖长日常甜蜜，将【玩家】与【主角】绑定，代入校园恋爱。
- **中期**：【B】失踪，情感锚点转为"她怎么了？我是受害者"的共情。
- **中后期**：【玩家】逐渐发现【主角】的异常，开始怀疑"我们的爱，是不是从一开始就错了？"
- **后期**：打破第四面墙（详见【叙事手法】的 Meta 终局），将【玩家】拖入责任链。

## 最终结局

与【玩家】对话的是【主角】内心创造出的理想化"她"——【C位】（由【E】与理想化投射融合而成）。她对【玩家】发出邀请："What's the time?" 游戏回到最初的输入框，变成一个开放的循环。

## 设计原则

1. **不要设计"玩家体验线"。** 去创造一堆记忆场景，每一个对应一个时间输入。【玩家】自己会成为那条线。
2. **记住疼痛，而非诡计。** 如果通关后【玩家】觉得"我真聪明，解开了复杂谜题"而不是"我说不出话"——那就失败了。在写作的每一天，都回到 03:15 那个黑暗的房间坐一会儿。那是所有故事开始的地方。
