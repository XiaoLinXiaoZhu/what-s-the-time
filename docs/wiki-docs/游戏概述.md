# What's the time — 游戏概述

## 类型

多周目文字冒险。外壳为轻松的校园恋爱喜剧，内核是预谋杀人、记忆扭曲与【叙事诡计】。

## 核心机制

每个周目开头都会出现类似的自问——「时间是……？」「不，是他在问我。」

【玩家】自由输入时间（HH:MM），游戏据此播放对应的【记忆的房间】。

## 项目愿景

打造一部文学性的、拒绝提供简单道德答案的心理惊悚文字冒险游戏。其最终体验并非解谜的快感或情感宣泄，而是类似于读完《人间失格》后，那种对人性深渊的凝视所带来的、无法言说且挥之不去的压抑与虚无感。

## 核心真相（玩家不可见）

- **两次杀人，同名受害者**：【主角】一年前在校园楼顶于【10:37】杀死恋人【A】；现在又将同名女生【B】视作替代品再次杀害。
- **栽赃与自我受害叙事**：【A案】中将责任栽赃给【H】；【B案】中复用同一套"故事模板"。
- **代偿人格与"她"的模糊身份**：杀死【A】后，【主角】在内心形成理想化的"她"（即【C位】的概念），后投射在【B】身上。
- **时间与记忆的扭曲**：真正卡住【主角】的是第一次杀人那一刻的【10:37】。多周目是他反复重播、剪辑记忆的过程。

## 最终形态轮廓

- **前期**：拖长日常甜蜜，将【玩家】与【主角】绑定。情感锚点是"和【B】的情感"。
- **中期**：情感锚点转为"【B】怎么死了？我是受害者"的共情路径。
- **中后期**：【玩家】逐渐发现【主角】的异常，情绪转变为"我们的爱，是不是从一开始就错了？"
- **后期**：【Meta突破】让【玩家】扮演【主角】执行杀人，与【C位】跳出游戏框架。

## 最终结局

与【玩家】对话的是【主角】内心创造出的理想化"她"——【C位】（由【E】与理想化投射融合而成）。她对【玩家】发出邀请："What's the time?" 游戏回到最初的输入框，变成一个开放的循环。
