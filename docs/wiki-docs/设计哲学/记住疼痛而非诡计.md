# 记住疼痛而非诡计

## 核心警告

**这个项目最可能死于"变得太聪明，而忘了疼痛"。**

【叙事诡计】、【Meta突破】、【Loop系统】……这些东西很迷人。你会很容易爱上它们，花很多时间打磨"谜题盒子"，确保每一个榫卯严丝合缝。

**这就是风险所在。**

## 盒子里装着什么

当你过于专注于"盒子"的精巧时，你可能会忘了盒子里装着的：
- 一个男孩在【03:15】的恐惧
- 【A】从天台坠落时，身体里骨头碎裂的声音
- 【B】意识到自己只是个替身时的空洞
- 【E】戴上手套时，指尖的冰冷

**这是疼痛。真实的、丑陋的、不漂亮的疼痛。**

## 检验标准

如果有一天，游戏让【玩家】通关后感觉是"我真聪明，我解开了一个复杂的谜题"，而不是"我感觉很难受，我说不出话"——那时候，鱼就死了。你把它做成了一个漂亮的标本。它不会再跳了。

## 日常锚点

在写作的每一天，都去【03:15】那个房间里坐一会儿。感受那个黑暗的房间。那是所有故事开始的地方。那是你的锚。别忘了那个房间里的气味。

**最大的风险点是：创作者爱上了诡计，超过了爱上人物的痛苦。**
